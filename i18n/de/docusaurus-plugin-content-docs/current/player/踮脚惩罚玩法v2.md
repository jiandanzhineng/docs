# Zehenspitzen-Belohnungsspiel v2

Hightech-Sensorisches Selbstdisziplin-Trainingsspiel - Zehenspitzen-Standherausforderungen

### V1.4.0 Update-Inhalte:
1. **TD01-Gerät hinzugefügt**: Verhindert lange Langeweile, startet nach einer bestimmten zeitspanne ohne Stromstoßauslösung mit allmählich zunehmender Intensität
2. **Schaltflächenstart für Zidongsuo hinzugefügt**: Erleichtert die Einzelbedienung. Wird nach Aktivierung des manuellen Startmodus das Spiel per Tastendruck gestartet
3. **Zidongsuo und TD01 sind optionale Geräte**: Das Spiel kann auch normal ohne diese beiden Geräte gespielt werden, nur Stromstoßgerät und QTZ-Sensor sind erforderlich

## 🎮 Detaillierte Spielanleitung
### 📋 Spielvorbereitung
#### 1. Gerätekonfiguration
Das System verbindet automatisch vier Kernkomponenten:

+ 🔒 **Automatische Verriegelung**: Wird zu Spielbeginn verriegelt, um vorzeitiges Verlassen zu verhindern (optionales Gerät)
+ ⚡ **Stromstoßgerät**: Liefert sofortige Rückmeldung und Bestrafung
+ 📡 **QTZ-Lasersensor**: Überwacht über zwei Tasten, ob die Ferse den Boden berührt (wird unter der Ferse platziert)
+ 🎮 **TD01-Gerät**: Optionale Erweiterung, startet nach einer bestimmten Zeitspanne ohne Stromstoßauslösung, erhöht allmählich die Intensität und stört das Zehenspitzen.

#### 2. Parameter einstellen
Die Spielschwierigkeit kann angepasst werden:

+ **Spieldauer**: Einstellbar von 1-360 Minuten
+ **Stromstoßintensität**: 10-100 Stufen
+ **Progressiver Modus**: Je häufiger Verstöße, desto stärker die Bestrafung
+ **Sicherheitsgrenze**: Optional, ob die Überschreitung der 30V-Sicherheitsgrenze erlaubt ist
+ **Manueller Start**: Optional, ob auf einen Tastendruck des Auto-Lock-Geräts gewartet werden muss, um das Spiel zu starten
+ **TD01-Verzögerung**: 1-60 Sekunden einstellbar. Wie lange nach Ausbleiben eines Stromstoßes wird TD01 gestartet
+ **TD01-Intensitätssteigerung**: Der pro 5 Sekunden hinzukommende Intensitätswert (1-100)

## 🎯 Konkrete Spielweise
### Kernregeln
Halten Sie die Pose auf den Zehenspitzen, die Ferse darf den Boden absolut nicht berühren!

#### 1. Spielbeginn
+ System verriegelt automatisch (falls Auto-Lock-Gerät vorhanden), vorzeitiges Beenden ist unmöglich
+ Bei aktiviertem manuellem Startmodus muss der Auto-Lock-Taster gedrückt werden, um den Timer zu starten
+ Timer startet, Zehenspitzenhaltung muss beibehalten werden
+ QTZ-Sensor beginnt mit der Echtzeitüberwachung der Fersenposition

#### 2. Überwachungsmechanismus
+ Der Sensor verfügt über zwei Tastenerfassungspunkte (button0 und button1)
+ Sobald die Ferse eine der beiden Tasten drückt = Verstoß
+ Das System löst sofort eine Stromstoßstrafe aus
+ Der Stromstoß stoppt nur, wenn beide Tasten nicht mehr gedrückt werden

#### 3. Bestrafungssystem
+ 🔴 **Sofortige Bestrafung**: Stromstoß beginnt im Moment, in dem die Ferse die Taste drückt
+ 📈 **Progressive Intensität**: Je mehr Verstöße, desto höher die automatisch ansteigende Stromstoßintensität
+ ⏱️ **Bestrafungsdauer**: Der Stromstoß dauert an, solange die Taste gedrückt ist, bis die Ferse vollständig gehoben wird
+ 🎮 **TD01-Erweiterungsstrafe**: Das TD01-Gerät startet nach einer festgelegten Zeit ohne ausgelösten Stromstoß. Intensität steigt alle 5 Sekunden

#### 4. TD01-Gerätemechanismus
+ **Verzögerter Start**: Startet automatisch nach der angegebenen Sekundenzahl ohne ausgelösten Stromstoß
+ **Intensitätssteigerung**: Automatische Erhöhung des eingestellten Intensitätswerts alle 5 Sekunden
+ **Maximale Intensität**: Unterstützt 0-255-Bereich, nicht durch Sicherheitsgrenze beeinflusst
+ **Automatischer Stopp**: Wird automatisch gestoppt und zurückgesetzt, sobald ein Stromstoß ausgelöst wird

#### 5. Spielende
+ Nach Erreichen der eingestellten Zeit wird automatisch entsperrt
+ Das System zeigt einen detaillierten Leistungsbericht an
+ Stellt Verstoßanzahl, Stromstoßanzahl und weitere Daten zusammen

## 💪 Herausforderungen im Überblick
+ **Körperliche Herausforderung**: Langes Stehen auf Zehenspitzen testet die Ausdauer der Beinmuskeln
+ **Willensprüfung**: Auch bei Ermüdung muss die korrekte Haltung gehalten werden
+ **Präzise Kontrolle**: Jegliches noch so geringes Absinken der Ferse wird erfasst
+ **Psychologische Auseinandersetzung**: Ruhe und Konzentration trotz Stromstoßandrohung bewahren
+ **Doppelter Druck**: Sowohl Stromstöße vermeiden, als auch mit Störungen durch das TD01-Gerät umgehen

## 🏆 Siegbedingungen
Erfolgreich bis zum Ablauf der festgelegten Zeit durchhalten. Je weniger Verstöße, desto besser die Leistung:

+ 🌟 **Perfekte Leistung**: 0 Verstöße
+ ⭐ **Ausgezeichnete Leistung**: 1-2 Verstöße
+ 👍 **Gute Leistung**: 3-5 Verstöße
+ 😅 **Verbesserungsbedarf**: Mehr als 5 Verstöße

## ⚠️ Sicherheitshinweise
+ Stellen Sie sicher, dass Sie gesund sind, also keine Herz-, Epilepsie- oder ähnliche Erkrankungen haben
+ Es wird empfohlen, mit niedriger Intensität und kurzer Zeit zu beginnen
+ Bei Unwohlsein während des Spiels sofort aufhören
+ Nur für Erwachsene über 18 Jahre
+ Seien Sie bei der Einstellung der Stromstoßintensität vorsichtig, zu hohe Intensität kann zu Stromstoßverletzungen führen. Das Risiko liegt bei Ihnen.
+ Wenn die Überschreitung der Sicherheitsgrenze aktiviert ist, erfolgt dies auf eigenes Risiko
+ Sorgen Sie für eine sichere Spielumgebung. Es empfiehlt sich Beaufsichtigung.

## 🔧 Besondere Funktionen
### Manueller Startmodus
+ Nach dem Spiel-Laden wird eine Warteoberfläche angezeigt
+ Auto-Lock-Gerät wird zuerst verriegelt
+ Der Auto-Lock-Taster muss gedrückt werden, um den Spiel-Timer zu starten
+ Geeignet für Situationen, die Vorbereitungszeit erfordern

### Echtzeit-Stromstoßmechanismus
+ Bei Drücken einer beliebigen Taste beginnt sofort der anhaltende Stromstoß
+ Der Stromstoß stoppt sofort, wenn alle Tasten losgelassen werden
+ Bietet ein direktes Feedback-System

### TD01-Erweiterungssystem
+ Optionale zusätzliche Herausforderung
+ Erhöht die Schwierigkeit bei guter Leistung
+ Intensität steigt allmählich, erhöht die Spannung im Spiel
+ Unterstützt unabhängige Intensitätskontrolle
