# Tippen Sie auf den Fußballen Spiel

## 🎭 Das faulende Hausangestellte-Strafespiel v1.4.0
Ein Hightech-Körpersensor-basiertes Selbstdisziplin-Trainingsspiel – Die Tippen-auf-den-Zehen-stehen-Herausforderung

### 🎮 Detaillierte Spielregeln 📋 Spielvorbereitung
1.  Gerätekonfiguration: Das System verbindet automatisch drei Kerngeräte
    - 🔒 Auto-Sperre: Verriegelt sich zu Spielbeginn, um vorzeitiges Beenden zu verhindern [Einfacher intelligenter Sperr-Terminal](../device/简单智能锁定终端.md)
    - ⚡ Elektroschockgerät: Bietet sofortige Feedback-Bestrafung [Einfacher intelligenter Impuls- (Schock-) Terminal](../device/简单智能脉冲（电击）终端.md)
    - 📡 QTZ Lasersensor: Überwacht über eine Fußplatte, ob die Ferse den Boden berührt (unter der Ferse platziert) [Einfacher intelligenter Entfernungssensor-Terminal](../device/简单智能距离感应终端.md)
+ **<font>Klicken Sie hier, um ein Geräteset zu kaufen:</font>** [Wer nicht auf den Zehen steht, wird geschockt - Spielset](https://item.taobao.com/item.htm?id=945819055827)
+  Parameter einstellen: Spezieller Spielschwierigkeitsgrad einstellbar
    - Spieldauer: Von 1 bis 180 Minuten einstellbar
    - Schockintensität: 10-100 Stufen
    - Eskalationsmodus: Je mehr Regelbrüche, desto stärker die Strafe 🎯 Genauer Ablauf

Kernregel: Halten Sie die Haltung auf den Zehenspitzen, die Ferse darf den Boden **niemals** berühren!

1.  Spielbeginn
    - System verriegelt automatisch, kann währenddessen nicht beendet werden
    - Zeitmessung beginnt, Haltung auf den Zehen muss beibehalten werden
    - QTZ-Sensor beginnt mit der Echtzeit-Überwachung der Fersenposition
2.  Überwachungsmechanismus
    - Der Sensor hat zwei Knopf-Erkennungspunkte
    - Sobald die Ferse einen beliebigen Knopf niederdrückt = Regelverstoß
    - System löst sofort Elektroschockstrafe aus
    - Der Schock hört erst auf, wenn **beide** Knöpfe nicht mehr gedrückt sind
3.  Strafensystem
    - 🔴 Sofortige Strafe: Der Schock beginnt im Moment, in dem die Ferse den Knopf niederdrückt
    - 📈 Eskalierende Intensität: Je mehr Regelverstöße, desto höher steigt die Schockstärke automatisch
    - ⏱️ Kontinuierliche Strafe: Solange ein Knopf gedrückt ist, hält der Schock an, bis er vollständig losgelassen wird
    - 🎮 Dilemma-Strafe: Nach einer Weile auf den Zehen startet TD01-Gerät zur Ablenkung
4.  Spielende
    - Nach Ende der eingestellten Zeit wird automatisch entriegelt
    - System zeigt einen vollständigen Leistungsbericht an
    - Stellt Statistik zu Regelverstößen, Schockanzahl etc. zusammen 💪 Herausforderungs-Schwerpunkte
+ Körperliche Herausforderung: Langes Stehen auf den Zehenspitzen stellt die Ausdauer der Beinmuskulatur auf die Probe
+ Selbstbeherrschungsprüfung: Trotz Ermüdung muss die korrekte Haltung gehalten werden
+ Präzise Kontrolle: Jedes noch so leichte Absenken der Ferse wird erkannt
+ Psychologisches Spiel: Ruhe und Konzentration trotz der Schock-Androhung bewahren 🏆 Siegbedingungen  
Erfolgreich bis zum eingestellten Zeitende durchhalten, je weniger Regelverstöße, desto besser die Leistung:
+ 🌟 Perfekte Leistung: 0 Regelverstöße
+ ⭐ Hervorragende Leistung: 1-2 Regelverstöße
+ 👍 Gute Leistung: 3-5 Regelverstöße
+ 😅 Verbesserung benötigt: Mehr als 5 Regelverstöße

### ⚠️ Sicherheitshinweis
+ Stellen Sie sicher, dass Sie bei guter Gesundheit sind und keine Krankheiten wie Herzkrankheiten haben
+ Es wird empfohlen, mit niedriger Intensität und kürzerer Dauer zu beginnen
+ Spielen Sie nicht bei Unwohlsein während des Spiels
+ Nur für Erwachsene über 18 Jahre
