# Erweiterte Version des Cunzhi-Spiels (3-Phasen-Version)

## Einführung
Dies ist ein Cunzhi-Trainingsspiel basierend auf einem Drucksensor. Das System passt die Vibrationsstärke intelligent basierend auf dem Druckwert an, um den Benutzer an der Erregungskante zu halten. Wenn der Druck den kritischen Wert überschreitet, wird eine Elektroschock-Bestrafung ausgelöst.

### Erklärung der Begriffe

Luftdruck = Drucksensor + T-Stück + 2 Schlauchstücke + aufblasbarer Anal-Stöpsel

Vibration = Eierkontroller + normale Größe von Liebeskugel/Ei

Elektroschock = Elektroschock-Terminal + Elektroden-Pads

Verriegelung = Automatische Verriegelung

Einzelstück für Männer = kleines Liebesei + Penisschutzhülle (siehe Bild unten)

Einzelstück für Frauen = Sexmaschine + Sexmaschinen-Controller

## Spielablauf
Das Spiel umfasst die folgenden zyklischen Zustände:

![Ablaufdiagramm](../img/cunzhi3phase.jpg)

1.  **Ruhephase (Initial/Sub Calm)**
    *   Die Vibrationsstärke steigt langsam an, begleitet von zufälligen Störungen.
    *   **Ziel**: Erregung steigern, Druck erhöhen.
    *   **Übergang**: Überschreitet der Druck den "mittleren Druck", tritt die Mittlere Phase ein.

2.  **Mittlere Phase (Middle)**
    *   Intelligente Steuerung: Je höher der Druck, desto niedriger die Vibrationsstärke (um ein zu schnelles Erreichen des Höhepunkts zu verhindern); je niedriger der Druck, desto höher die Vibrationsstärke (um die Erregung aufrechtzuerhalten).
    *   **Übergang**:
        *   Druck >= "kritischer Druck": Löst **Elektroschock** aus und tritt in die Edging-Phase ein.
        *   Druck < "mittlerer Druck": Zurückfall in die Ruhephase.

3.  **Edging-Phase (Edging)**
    *   **Elektroschock wird ausgelöst**! Vibration stoppt.
    *   **Übergang**: Sinkt der Druck unter den kritischen Wert, tritt die Verzögerungsphase ein.

4.  **Verzögerungsphase (Delay)**
    *   Wartezeit (Abkühlverzögerung).
    *   **Übergang**: Nach Ende der Verzögerung wird basierend auf dem aktuellen Druck entschieden, ob zur Mittleren Phase zurückgekehrt wird (größer als mittlerer Druck) oder zur Ruhephase (kleiner als mittlerer Druck).



## Benötigte Geräte
1.  **Drucksensor (QIYA)**: Erforderlich. Zur Überwachung des Erregungsgrades.
2.  **Exzentermotor-Controller (TD01)**: Erforderlich. Zur Bereitstellung der Vibrationsstimulation.
3.  **Elektroschock-Gerät (DIANJI)**: Optional. Zur Bestrafung am Rand/Kurz vor dem Höhepunkt.
4.  **Automatische Verriegelung (ZIDONGSUO)**: Optional. Verriegelt automatisch zu Spielbeginn und entriegelt am Ende.


## Gebrauchsanweisung {#use_instruction}

1.  Montage der Druckvorrichtung abschließen.
    *   Videoanleitung zur Montage: https://youtu.be/QQaT823gKHc
    *   Quark: https://pan.quark.cn/s/03b8d2dab2ff
    *   Für die OSR6-Version siehe [OSR6-Spielanleitung](../../device/osr6-control-plugin.md)
2.  Alle Geräte mit dem Netzwerk verbinden
    *   Handy: Siehe [Mobile Client](../client/new-phone-client.md)
    *   Computer: Siehe [PC-Version des Steuerclients](../client/PC版控制客户端.md)

2.  Die Druckvorrichtung in den Körper einführen, normalerweise auf 20-25 kPa aufpumpen. Stellen Sie sicher, dass der Druck bei leichtem Druck deutlich spürbare Veränderungen aufweist und dabei angemessen niedrig ist, um ein Entweichen der Luft zu verlangsamen.
Hinweis: In den ersten 3 Minuten nach dem Start des Drucksensors kann es zu instabilen Druckverhältnissen kommen (kontinuierliches Absinken oder Ansteigen). Dies ist der Sensor im Aufwärmmodus. Bitte warten Sie in diesem Fall einen Moment, bevor Sie mit dem Spiel beginnen.
3.  Mittleren Druck und kritischen Druck einstellen. Beim ersten Spielen können Sie diese Werte während des Spiels anpassen. Verringern Sie die Werte, wenn es unerträglich wird, erhöhen Sie sie, wenn keine Empfindung spürbar ist.


## Konfigurationsbeschreibung

| Parametername | Beschreibung | Empfohlener Wert |
| :--- | :--- | :--- |
| **Spieldauer** | Gesamtlaufzeit des Spiels (Minuten). | 20 |
| **Kritischer Druck** | Höchster Druck-Schwellenwert zum Auslösen von Elektroschock und Cunzhi (kPa). | 23 |
| **TD01 maximale Stärke** | Maximale Stärke der Vibration (1-255). | 255 |
| **Mittlerer (Erregungs-)Druck** | Druck-Schwellenwert für den Eintritt in die intelligente Steuerungsphase (kPa). | 22 |
| **Elektroschock-Stärke** | Spannungsstärke während der Bestrafung (V). | 20 |
| **Elektroschock-Dauer** | Dauer der Bestrafung (Sekunden). | 3 |
| **Niederdruck-Verzögerung** | Abkühl-/Ruhezeit nach Auslösung von Cunzhi (Sekunden). | 5 |
| **Maximale Stärkesteigerungsrate** | Höchstgeschwindigkeit der Stärkeerhöhung der Vibration, um abrupte Veränderungen zu vermeiden. | 10 |
| **Stärke-Zufallsstörung** | Zufällige Schwankungsamplitude der Vibrationsstärke (%). | 0 |
| **Schrittweise Erhöhung nach Verzögerung** | Wert, um den die Vibration in der Ruhephase pro Sekunde automatisch erhöht wird. | 10 |
