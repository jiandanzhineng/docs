# Bedienungsanleitung für den PC-Client

1. Download-Link: [https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip](https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip)
2. **Bitte entpacken Sie die Datei und installieren Sie das Programm (falls das Entpacken fehlschlägt, können Sie [7zip](https://www.7-zip.org/download.html) installieren und die Datei mit dieser Software entpacken**)**
3. **Starten Sie den Client**
4. **Aktivieren Sie den Computer-Hotspot, setzen Sie den WLAN-Namen auf »easysmart«, das Passwort auf acht Einsen und das Band auf ****2,4 GHz**** (bei Problemen: [Nutzungsweise ohne Computer-Hotspot](#无电脑热点情况使用方式))**  
Hinweis: Wenn das Gerät bereits mit einem anderen 5-GHz-WLAN verbunden ist, kann es sein, dass es trotz 2,4-GHz-Konfiguration weiterhin im 5-GHz-Band arbeitet. In diesem Fall muss das Gerät mit einem 2,4-GHz-WLAN (oder per Ethernet) verbunden werden, um den WLAN-Hotspot richtig einrichten zu können.
5. **Wenn das Gerät mit dem WLAN verbunden ist, aber im Client nicht angezeigt wird, wurde es möglicherweise blockiert. Deaktivieren Sie in diesem Fall die Firewall auf dem Computer.**
6. **Starten Sie das Gerät. Nach etwa 15–30 Sekunden verbindet es sich automatisch mit dem Computer.**
7. **Nun können Sie loslegen und Spaß haben!**

 

## Nutzungsweise ohne Computer-Hotspot {#无电脑热点情况使用方式}
1. Gerätevernetzung: Bitte lesen Sie [Gerät mit WLAN verbinden (Vernetzung) über die APP](../../other/设备连接wifi（配网）/通过APP将设备连接到wifi.md), um das Gerät mit Ihrem Heim-WLAN zu verbinden (beachten Sie, dass es sich um ein 2,4-GHz-WLAN handeln muss).
2. Aktivieren Sie mDNS im Client.
