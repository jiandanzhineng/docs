# Anleitung zur Verwendung des neuen PC-Clients

1.  Laden Sie [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip) herunter.
2.  **<font>Bitte entpacken Sie die Datei vor der Installation. Wenn das Entpacken fehlschlägt, installieren Sie</font>** [7zip](https://www.7-zip.org/download.html) **<font>und entpacken Sie die Datei anschließend mit dieser Software.</font>**
3.  **<font>Starten Sie den Client.</font>**
4.  **<font>Aktivieren Sie den PC-Hotspot, setzen Sie den WLAN-Namen auf 'easysmart', das Passwort auf acht Einsen (11111111) und das Frequenzband auf das</font>****<font>2,4-GHz</font>****<font>-Band (falls kein PC-Hotspot verfügbar ist, siehe weiter unten).</font>**
5.  **<font>(Wichtig)</font>****<font>Wenn das Gerät mit dem WLAN verbunden ist, aber im Client nicht angezeigt wird, wurde es möglicherweise blockiert. Deaktivieren Sie in diesem Fall die Windows-Firewall.</font>**
6.  **<font>Starten Sie das Gerät. Etwa 15-30 Sekunden nach dem Start verbindet es sich automatisch mit dem PC.</font>**
7.  **<font>Jetzt kann der Spaß beginnen!</font>**

## Verwendung ohne PC-Hotspot

1.  Gerätekonfiguration für das Netzwerk: Befolgen Sie die Anleitung unter [Gerät über APP mit WiFi verbinden](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md), um das Gerät mit Ihrem Heim-WiFi zu verbinden (Achtung: Es muss ein 2,4-GHz-WLAN sein).
2.  Aktivieren Sie mDNS im Client.
