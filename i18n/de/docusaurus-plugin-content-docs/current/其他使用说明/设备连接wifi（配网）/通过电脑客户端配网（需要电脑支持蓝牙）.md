# Netzwerkkonfiguration über den PC-Client (erfordert Bluetooth-Unterstützung am PC)

<font>Hinweis: Diese Methode wird noch getestet und ist möglicherweise instabil. Bei einem Fehlschlag können Sie versuchen, den Konfigurationsvorgang zu beenden und neu zu starten oder eine andere Methode zu verwenden.</font>

**<font>Hinweis: Wenn Sie das Standard-WLAN verwenden möchten, führen Sie bitte keine Netzwerkkonfiguration durch. Der Standard-WLAN-Name lautet 'easysmart' und das Passwort ist '11111111'. Das Gerät verbindet sich automatisch mit dem Standard-WLAN.</font>**

![1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4.png](./img/hURz820DTNcJYUCH/1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4-096925.png)

1.  Öffnen Sie den Client, klicken Sie auf "Geräteverwaltung" und dann auf "Netzwerkkonfiguration".

![1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed.png](./img/hURz820DTNcJYUCH/1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed-155975.png)

2.  Geben Sie den WLAN-Namen und das WLAN-Passwort ein (achten Sie darauf, keine Fehler einzugeben, da sonst nach erfolgreicher Konfiguration keine Verbindung zum Internet hergestellt werden kann).
3.  Klicken Sie auf "Konfiguration ändern".
4.  Klicken Sie auf "Netzwerkkonfiguration starten", um den Vorgang zu beginnen.
