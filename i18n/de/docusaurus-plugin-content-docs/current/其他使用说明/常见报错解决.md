# Häufige Fehler und ihre Lösungen

1. Das Gerät kann nach der Konfiguration des Netzwerks nicht automatisch online gehen, wenn der Client gestartet wird
    1. Bitte überprüfen Sie, ob Ihr Router mdns unterstützt. Weitere Informationen finden Sie unter [Überprüfen, ob der Router mdns unterstützt (die meisten Router unterstützen mdns)](./检测路由器是否支持mdns（大部分路由器都支持）.md). Wenn festgestellt wird, dass mdns nicht unterstützt wird, können Sie
        1. Computer und Gerät über einen mobilen Hotspot verbinden, um zu spielen
        2. Die Verbindungsbeziehung zwischen Ihrem Router und Ihrem Modem überprüfen. Wenn der Router im AP-Modus betrieben wird, können Sie ihn in den DHCP-Modus ändern

2. Fehler im Zusammenhang mit vcruntime140.dll

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

Laden Sie das <font>Visual C++ Redistributable for Visual Studio 2015</font> herunter und installieren Sie es.
