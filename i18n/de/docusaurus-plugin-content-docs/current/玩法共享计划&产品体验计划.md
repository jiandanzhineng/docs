# Spiel-Sharing-Programm & Produkt-Erfahrungsprogramm

# Spiel-Sharing-Programm  
Willkommen an Benutzer, die auf der Basis des Geräts eigene Spielweisen entwickeln.

Nach Abschluss der Entwicklung können Sie sich mit dem Up-Communicator in Verbindung setzen, um das entsprechende Spielset aufzulisten. Nach dem Verkauf erhalten die Spielentwickler eine 5 %-ige Umsatzbeteiligung.

Für Details können Sie sich an den Bilibili-Up, den Taobao-Shop-Besitzer oder den QQ-Gruppen-Administrator wenden.

# Produkt-Erfahrungsprogramm  
1. Wählen Sie die Spielweise aus, die Sie erleben möchten.  
2. Füllen Sie das Anmeldeformular für Product Experience Officer aus: [https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS](https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS)  
3. Bei Bedarf wird sich der Kundenservice mit Ihnen in Verbindung setzen.  
4. Bestellen Sie die Ware.  
5. Machen Sie ein Nutzungsvideo und senden Sie es.  
6. Der Warenbetrag wird Ihnen erstattet.

Einige Hinweise: Es ist nicht notwendig, private Bereiche preiszugeben.
