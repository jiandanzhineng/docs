# Spielanleitungs-Sharing-Programm & Produkterfahrungs-Programm

# Spielanleitungs-Sharing-Programm
Nutzer sind eingeladen, basierend auf dem Gerät eigene Spielanleitungen zu entwickeln.

Nach Abschluss der Entwicklung können Sie sich an den Up-Hersteller wenden, um das entsprechende Spielanleitungsset in den Shop aufzunehmen. Nach dem Verkauf erhalten der Entwickler der Spielanleitung 5 % des Verkaufserlöses als Anteil.

Für Details können Sie sich an den Bilibili-Up-Hersteller, den Taobao-Shop-Betreiber oder den QQ-Gruppen-Administrator wenden.

# Produkterfahrungs-Programm
1. Wählen Sie die Spielanleitung aus, die Sie ausprobieren möchten.
2. Füllen Sie das Anmeldeformular für Produkttester aus [https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS](https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS)
3. Bei Bedarf wird Sie der Kundenservice kontaktieren.
4. Bestellen Sie das Produkt.
5. Nehmen Sie ein Nutzungsvideo auf und senden Sie es.
6. Der Produktbetrag wird Ihnen überwiesen.

Einige Hinweise: Es ist nicht erforderlich, private Körperbereiche zu zeigen.
