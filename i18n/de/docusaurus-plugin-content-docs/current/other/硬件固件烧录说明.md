# Anleitung zum Flash-Firmware-Update für Hardware

1. Treiber installieren
    1. CH340 UART-Chip-Treiber: [https://www.wch.cn/downloads/CH341SER_EXE.html](https://www.wch.cn/downloads/CH341SER_EXE.html)
    2. CH343 UART-Chip-Treiber: [https://www.wch.cn/downloads/CH343SER_EXE.html](https://www.wch.cn/downloads/CH343SER_EXE.html)
    3. Nach dem Download einfach ausführen
2. Flash-Tool herunterladen
    1. [https://dl.espressif.com/public/flash_download_tool.zip](https://dl.espressif.com/public/flash_download_tool.zip)
    2. Entpacken und ausführen
    3. Bedienungsanleitung:
    4. [https://docs.espressif.com/projects/esp-test-tools/zh_CN/latest/esp32/production_stage/tools/flash_download_tool.html#id5](https://docs.espressif.com/projects/esp-test-tools/zh_CN/latest/esp32/production_stage/tools/flash_download_tool.html#id5)
    5. Im Hinweis in Schritt 3 muss der COM-Port angepasst werden. Andere Teile bleiben unverändert.
3. Firmware herunterladen
    1. [https://github.com/jiandanzhineng/hardware/releases/latest](https://github.com/jiandanzhineng/hardware/releases/latest)
    2. ![1768981515944-c052b509-6ae3-41d5-b212-367666940cfd.png](./img/IpJdu5RTAcw-LC13/1768981515944-c052b509-6ae3-41d5-b212-367666940cfd-449925.png)
    3. Firmware je nach Gerätetyp herunterladen (Firmware mit dem Hinweis "merged" herunterladen) Elektroschock-DIANJI, Drucksensor-QIYA, Taumelscheiben Motorsteuerung-TD01, Abstandssensor-QTZ
    4. Mit dem Tool auf Geräte an Adresse 0x0 flashen.
