# 新版PCクライアント使用ガイド

1. [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip) をダウンロード
2. **<font>解凍してからインストールしてください。解凍に失敗した場合は、</font>**[7zip](https://www.7-zip.org/download.html)**<font>をインストール後、そのソフトで解凍してください。</font>**
3. **<font>クライアントを起動</font>**
4. **<font>PCのモバイルホットスポットを有効にし、WiFi名を「easysmart」、パスワードを「11111111」（8つの1）、周波数帯を</font>****<font>2.4GHz</font>****<font>に設定してください（PCにホットスポット機能がない場合は後述の手順を参照）。</font>**
5. **<font>（重要）</font>****<font>デバイスがWiFiに接続されているのにクライアントに表示されない場合は、遮断されている可能性があります。PCのファイアウォールを無効にしてください。</font>**
6. **<font>デバイスを起動します。起動後、約15～30秒でPCに自動接続されます。</font>**
7. **<font>これで楽しくご利用いただけます。</font>**

 

## PCにホットスポット機能がない場合の使用方法
1. デバイスのネットワーク設定：[APP経由でデバイスをWiFiに接続する](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) を参考に、デバイスを自宅のWiFi（必ず2.4GHz WiFiであること）に接続してください。
2. クライアントでmDNSを有効にします
