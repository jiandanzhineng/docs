# 新バージョン パソコン クライアント使用ガイド

1. [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip) をダウンロード
2. **<font>解凍後にインストールしてください。解凍に失敗した場合は、</font>**[7zip](https://www.7-zip.org/download.html)**<font> をインストールしてからそのソフトウェアで解凍してください。</font>**
3. **<font>クライアントを起動する</font>**
4. **<font>パソコンのホットスポットを有効にし、Wi-Fi名を easysmart、パスワードを8つの1（11111111）、周波数帯を</font>****<font>2.4G</font>****<font>帯に設定します（パソコンのホットスポットがない場合は後述を参照）。</font>**
5. **<font>（重要）</font>****<font>デバイスがWi-Fiに接続しているのにクライアントにデバイスが表示されない場合は、ブロックされている可能性があります。パソコンのファイアウォールをオフにしてください。</font>**
6. **<font>デバイスを起動します。デバイス起動後、約15～30秒で自動的にパソコンに接続されます。</font>**
7. **<font>これで楽しく遊べます！</font>**

 

## パソコンのホットスポットがない場合の使用方法
1. デバイスのネットワーク設定：[APP を使用してデバイスを Wi-Fi に接続する](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) を参考に、デバイスを自宅の Wi-Fi に接続します（必ず 2.4G Wi-Fi であることに注意）。
2. クライアントで mDNS を有効にします。
