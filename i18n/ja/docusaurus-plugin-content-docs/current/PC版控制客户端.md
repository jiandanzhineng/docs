# PCクライアント使用ガイド

1.  ダウンロードリンク：[https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2.  **ダウンロード後、解凍してインストールしてください。解凍に失敗した場合は、[7zip](https://www.7-zip.org/download.html)をインストールしてからそのソフトウェアで解凍してください。**
3.  **クライアントを起動する**
4.  **PCのモバイルホットスポットを有効にし、WiFi名を`easysmart`、パスワードを`11111111`（8つの1）、バンドを**2.4G**に設定してください（[PCのモバイルホットスポットがない場合の使用方法](#pcのモバイルホットスポットがない場合の使用方法)）**
5.  **（重要）デバイスがWiFiに接続しているのにクライアントにデバイスが表示されない場合は、ブロックされている可能性があります。PCのファイアウォールをオフにしてください。**
6.  **デバイスの電源を入れます。デバイス起動後、約15〜30秒で自動的にPCに接続されます。**
7.  **これで楽しく遊べます！**

## PCのモバイルホットスポットがない場合の使用方法
1.  デバイスのネットワーク設定： [APP経由でデバイスをWiFiに接続する](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) を参照して、デバイスを自宅のWiFiに接続してください（**2.4G WiFi**である必要があります）。
2.  クライアントでmDNSを有効にする
