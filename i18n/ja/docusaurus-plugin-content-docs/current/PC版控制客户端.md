# 新版PCクライアント使用ガイド

1. [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip) をダウンロード
2. **<font>解凍してからインストールしてください。解凍に失敗した場合は、</font>**[7zip](https://www.7-zip.org/download.html)**<font>をインストール後、そのソフトウェアで解凍してください。</font>**
3. **<font>クライアントを起動</font>**
4. **<font>PCのホットスポットを有効にし、WiFi名をeasysmart、パスワードを8つの1、周波数帯を</font>****<font>2.4G</font>****<font>帯に設定してください（PCホットスポットがない場合は後述を参照）。</font>**
5. **<font>（重要）</font>****<font>デバイスがWiFiに接続しているのにクライアントにデバイスが表示されない場合は、ブロックされている可能性があります。PCのファイアウォールをオフにしてください。</font>**
6. **<font>デバイスを起動します。デバイス起動後、約15〜30秒で自動的にPCに接続されます。</font>**
7. **<font>楽しく使い始めましょう！</font>**

 

## PCホットスポットがない場合の使用方法
1. デバイスのネットワーク設定：[APP経由でデバイスをWiFiに接続する](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) を参考に、デバイスを自宅のWiFiに接続してください（必ず2.4G WiFiであることに注意）。
2. クライアントでmdnsを有効にする
