# 既知の問題

1. 一部のルーターではデバイスのネットワーク設定が成功しない場合があります。この場合、スマートフォンのWi-Fiをテザリングしてご利用ください。
2. デバイスを初期受取時に、ネットワーク設定の失敗や設定後のオンライン状態にならない場合があります。これはバッテリー切れが原因の可能性があります。充電するか電源に接続した状態で、再起動をお試しください。
3. ネットワーク状態が安定している環境でmDNSを起動してください。不安定な環境（例：スマートフォンのホットスポット接続時）では、先にホットスポットに接続してからmDNSを起動しないと、mDNSドメインのバインドに異常が発生する可能性があります。
4. スマートフォンがデバイスに接続した後、ネットワーク設定ボタンが灰色になる場合があります。この場合はスマートフォンを再起動してみてください。
5. ホットスポット接続モードでは、ネットワークカードの品質により、一定間隔で短時間の動作遅延が発生する可能性があります。
