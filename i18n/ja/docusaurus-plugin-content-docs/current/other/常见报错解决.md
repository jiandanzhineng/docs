# よくあるエラーの解決方法

1. デバイスをネットワークに設定した後、クライアント起動時に自動的にオンラインにならない
    1. ルーターが mdns をサポートしているか確認してください。参考：[ルーターが mdns をサポートしているか確認する（ほとんどのルーターはサポートしています）](./检测路由器是否支持mdns（大部分路由器都支持）.md) サポートしていない場合は：
        1. スマートフォンのホットスポットを使用してPCとデバイスを接続してプレイする
        2. ルーターと光回線モデムの接続関係を確認し、ルーターのモードがAPの場合はDHCPモードに変更する
2. vcruntime140.dll 関連のエラー

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

<font>Visual C++ Redistributable for Visual Studio 2015</font> をダウンロードしてインストールしてください

 

2. 起動時に以下のエラーが表示される場合

<font>Unable to load emulator DLL</font>  
<font>(C:\beidanci\emqx\erts-13.2.2.4\bin\beam.smp.dll)</font>  
<font>この時 SET は存在しません。</font>

 

<font>解決策：Erlangをインストールしてください</font>

[Downloads - Erlang/OTP](https://www.erlang.org/downloads)
