# PCクライアントを使用したネットワーク設定（PCのBluetooth対応が必要）

<font>注意：この方法はまだテスト中であり、不安定な可能性があります。失敗した場合は設定を終了して再開するか、他の方法をお試しください</font>

**<font>注意：デフォルトWi-Fiを使用したい場合はネットワーク設定を行わないでください。デフォルトWi-Fi名はeasysmart、パスワードは11111111です。デバイスは自動的にデフォルトWi-Fiに接続します</font>**

![1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4.png](./img/hURz820DTNcJYUCH/1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4-096925.png)

1. クライアントを開き、「デバイス管理」→「デバイスネットワーク設定」をクリック

![1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed.png](./img/hURz820DTNcJYUCH/1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed-155975.png)

2. Wi-Fi名とWi-Fiパスワードを入力（誤入力に注意。設定成功後もインターネットに接続できない場合があります）
3. 「設定変更」をクリック
4. 「ネットワーク設定開始」をクリックすると設定が開始されます
