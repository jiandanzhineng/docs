# シンプルスマートパルス（電気刺激）端末

購入リンク: [https://item.taobao.com/item.htm?id=892309919507](https://item.taobao.com/item.htm?id=892309919507)

注意事項：  
1**. デバイスはデフォルトでsafeモードになっており、このとき電圧は36V以下です。電圧を上げたい場合は、デバイスのsafe属性を0に設定してください。このモードでは電圧の上限は80Vとなりますが、ご自身で安全に注意してください。問題が発生した場合は、自己責任で対応してください。**

2. **コネクターを奥までしっかり挿入してください。挿入が不十分だと出力されません**
