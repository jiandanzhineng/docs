# 気圧寸止めプレイ3段階アップグレード版

## 概要
これは気圧センサーに基づいた寸止めトレーニングプレイです。システムは気圧値に応じて振動強度をインテリジェントに調整し、ユーザーを興奮の瀬戸際に導きます。気圧が臨界値を超えると、電気ショックによる罰則が発動します。

## ゲームフロー
ゲームは以下の状態サイクルで構成されます：

![フロー図](./img/cunzhi3phase.jpg)

1. **平静期 (Initial/Sub Calm)**
   - 振動強度がゆっくり上昇し、ランダムな揺らぎを伴います。
   - **目標**：興奮度を高め、気圧を上げる。
   - **遷移**：気圧が「中間圧力」を超えると中期へ。

2. **中期 (Middle)**
   - インテリジェント制御：気圧が高いほど振動強度は低く（急激な爆発を防ぐ）、気圧が低いほど振動強度は高く（興奮を維持）。
   - **遷移**：
     - 気圧 >= 「臨界気圧」：**電気ショック**を発動し、エッジング期へ。
     - 気圧 < 「中間圧力」：平静期へ戻る。

3. **エッジング期 (Edging)**
   - **電気ショック発動**！振動は停止。
   - **遷移**：気圧が臨界値以下に下がると、クールダウン遅延期へ。

4. **遅延期 (Delay)**
   - 一定時間待機（冷静化遅延）。
   - **遷移**：遅延終了後、現在の気圧に応じて中期（中間圧力より大きい場合）または平静期（中間圧力より小さい場合）へ戻る。

## 必要なデバイス
1. **気圧センサー (QIYA)**：必須。興奮度のモニタリング用。
2. **偏心モーターコントローラー (TD01)**：必須。振動刺激の提供用。
3. **電気ショックデバイス (DIANJI)**：オプション。エッジング時の罰則用。
4. **自動ロック (ZIDONGSUO)**：オプション。ゲーム開始時に自動ロック、終了時にアンロック。

## 使用方法

1.  気圧デバイスの組み立てを完了し、すべてのデバイスをネットワークに接続します。
2.  気圧デバイスを体内に入れ、通常は20-25kpaまで充気します。軽く押した時に圧力が明確に変化する状態で、かつ適度に低い圧力になるように調整します。これにより空気漏れを緩和できます。
3.  中間気圧と臨界気圧を設定します。初めてプレイする場合は、プレイしながら調整できます。この2つの気圧で我慢できなくなったら少し下げ、感じなければ少し上げます。

## 設定説明

| パラメータ名 | 説明 | 推奨値 |
| :--- | :--- | :--- |
| **ゲーム時間** | ゲームの総実行時間（分）。 | 20 |
| **臨界気圧** | 電気ショックと寸止めを発動する最高気圧閾値 (kPa)。 | 23 |
| **TD01最大強度** | 振動の最大強度 (1-255)。 | 255 |
| **中間(興奮)圧力** | インテリジェント制御段階に入る圧力閾値 (kPa)。 | 22 |
| **電気ショック強度** | 罰則時の電圧強度 (V)。 | 20 |
| **電気ショック持続** | 罰則の持続時間 (秒)。 | 3 |
| **低圧遅延** | 寸止め発動後のクールダウン休憩時間 (秒)。 | 5 |
| **強度増加速度上限** | 振動強度の最大上昇速度。急激な変化を防ぎます。 | 10 |
| **強度ランダム揺らぎ** | 振動強度のランダムな変動幅 (%)。 | 0 |
| **遅延後段階的向上** | 平静期における振動強度の毎秒自動増加値。 | 10 |
