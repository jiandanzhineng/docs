# つま先立ち罰ゲーム v2

ハイテク体感自律トレーニングゲーム - つま先立ちチャレンジ

### V1.4.0更新内容：
1.  **TD01デバイス追加**：長時間の退屈を防ぐため、電気ショックが一定時間発生しないと起動し、強度が徐々に増加します
2.  **自動ロックキー操作によるゲーム開始機能追加**：単独操作を容易にするため、手動開始モード有効時にキー操作でゲームを開始できます
3.  **自動ロック、TD01はオプションデバイス**：これらのデバイスがなくても、電気ショックデバイスとQTZセンサーだけで通常通りゲームをプレイできます

## 🎮 ゲームプレイ詳細
### 📋 ゲーム準備
#### 1. デバイス設定
システムは自動的に4つのコアデバイスを接続します：

+ 🔒 **自動ロック**：ゲーム開始時にロックされ、途中退出を防止します（オプションデバイス）
+ ⚡ **電気ショックデバイス**：即時フィードバックと罰を提供
+ 📡 **QTZレーザーセンサー**：2つのボタンでかかとが下がったかどうかを監視（かかとの下に設置）
+ 🎮 **TD01デバイス**：オプション強化デバイス。電気ショックが一定時間発生しないと起動し、強度が徐々に増加してつま先立ちを妨害します。

#### 2. パラメーター設定
ゲームの難易度をカスタマイズ可能：

+ **ゲーム時間**：1-360分で調整可能
+ **電気ショック強度**：10-100レベル
+ **段階的強化モード**：違反回数が多いほど罰が重くなります
+ **安全制限**：30Vの安全制限を超えることを許可するか選択可能
+ **手動開始**：自動ロックデバイスのキーがクリックされるのを待ってゲームを開始するか選択可能
+ **TD01遅延時間**：1-60秒で調整可能。電気ショックが発生しないまま何秒経過後にTD01を起動するか
+ **TD01強度増加量**：5秒ごとに増加する強度の値（1-100）

## 🎯 具体的なプレイ方法
### 基本ルール
つま先立ちの姿勢を維持し、かかとは絶対に地面に接触させない！

#### 1. ゲーム開始
+ システムが自動的にロック（自動ロックデバイスがある場合）、途中で退出できなくなります
+ 手動開始モードが有効な場合、自動ロックデバイスのキーをクリックしてタイマーを開始する必要があります
+ タイマー開始、つま先立ち姿勢を維持する必要があります
+ QTZセンサーがかかとの位置をリアルタイムで監視開始

#### 2. 監視メカニズム
+ センサーには2つのボタン検出点（button0とbutton1）があります
+ かかとがいずれかのボタンを押し下げた瞬間 = 違反
+ システムは直ちに電気ショックの罰を発動
+ 両方のボタンが押されていない状態になって初めて、電気ショックが停止します

#### 3. 罰システム
+ 🔴 **即時罰**：かかとがボタンを押し下げた瞬間に電気ショック開始
+ 📈 **段階的強化**：違反回数が増えるほど、電気ショック強度が自動的に増加
+ ⏱️ **持続的罰**：ボタンが押されている間は継続的に電気ショック、完全に離すまで続きます
+ 🎮 **TD01強化罰**：電気ショックが指定時間発生しないとTD01デバイスが起動し、強度が5秒ごとに増加

#### 4. TD01デバイスの仕組み
+ **遅延起動**：電気ショックが指定秒数発生しないと自動起動
+ **強度増加**：5秒ごとに設定された強度値が自動的に増加
+ **最大強度**：0-255の範囲をサポート、安全制限の影響を受けません
+ **自動停止**：電気ショックが発生すると自動的に停止しリセットされます

#### 5. ゲーム終了
+ 設定時間を堅持すると自動的にロック解除
+ システムが完全なパフォーマンスレポートを表示
+ 違反回数、電気ショック回数などのデータを統計

## 💪 チャレンジのポイント
+ **体力チャレンジ**：長時間のつま先立ちで脚の筋肉の持久力が試されます
+ **意志力の試練**：疲労に直面しても標準的な姿勢を維持する必要があります
+ **精密な制御**：かかとのわずかな沈み込みも検知されます
+ **心理的な駆け引き**：電気ショックの脅威のもとで冷静さと集中力を保ちます
+ **二重のプレッシャー**：電気ショックを避けながら、TD01デバイスの干渉にも対処する必要があります

## 🏆 勝利条件
設定時間終了まで成功裏に堅持し、違反回数が少ないほど優秀な成績です：

+ 🌟 **完璧なパフォーマンス**：0回の違反
+ ⭐ **優秀なパフォーマンス**：1-2回の違反
+ 👍 **良好なパフォーマンス**：3-5回の違反
+ 😅 **改善が必要**：5回以上の違反

## ⚠️ 安全上の注意
+ 心臓病、てんかんなどの疾患がないことを確認してください
+ 低強度・短時間から始めることをお勧めします
+ ゲーム中に体調不良を感じた場合は直ちに中止してください
+ 18歳以上の成人のみが使用可能です
+ 電気ショック強度は慎重に設定してください。高すぎると感電傷害の可能性があり、リスクは自己責任です
+ 安全制限を超える機能を有効にする場合、リスクは自己負担となります
+ ゲーム環境の安全を確保し、監督者を配置することをお勧めします

## 🔧 特徴的な機能
### 手動開始モード
+ ゲームのロード完了後、待機画面を表示
+ 自動ロックデバイスが先にロック
+ 自動ロックデバイスのキーをクリックしてゲームタイマーを開始する必要があります
+ 準備時間が必要なシナリオに適しています

### リアルタイム電気ショックメカニズム
+ いずれかのボタンが押された瞬間に継続的な電気ショックが開始
+ すべてのボタンが離された瞬間に電気ショックが停止
+ 最も直接的なフィードバックメカニズムを提供

### TD01強化システム
+ オプションの追加チャレンジデバイス
+ 良いパフォーマンス時に難易度を追加
+ 強度が徐々に増加し、ゲームの緊張感を高めます
+ 独立した強度制御をサポート
