# つま先立ちの罰ゲーム

## 🎭 メイドさんのサボり罰ゲーム v1.4.0
ハイテク体感自律トレーニングゲーム - つま先立ちチャレンジ

### 🎮 ゲームプレイの詳細 📋 ゲーム準備
1.  デバイス設定 ：システムは自動的に3つのコアデバイスを接続します
    *   🔒 自動ロック：ゲーム開始時にロックされ、途中退出を防止 [シンプルスマートロック端末](../设备介绍&购买/简单智能锁定终端.md)
    *   ⚡ 電気刺激デバイス：即時フィードバックの罰を提供 [シンプルスマートパルス（電気刺激）端末](../设备介绍&购买/简单智能脉冲（电击）终端.md)
    *   📡 QTZレーザーセンサー：足踏みプレートを通じてかかとが下がっているかを監視（かかとの下に設置）[シンプルスマート距離感知端末](../设备介绍&购买/简单智能距离感应终端.md)
2.  **<font>ここをクリックしてデバイス一式を購入：</font>**[つま先を上げないと電気刺激セット](https://item.taobao.com/item.htm?id=945819055827)
3.  パラメーター設定 ：ゲームの難易度をカスタマイズ可能
    *   ゲーム時間：1～180分で調整可能
    *   電気刺激強度：10～100レベル
    *   進行モード：違反回数が増えるほど、罰が重くなる 🎯 具体的なプレイ方法

**核心ルール** ：つま先立ちの姿勢を保ち、かかとを絶対に地面につけてはいけない！

1.  ゲーム開始
    *   システムが自動的にロックされ、途中で退出できなくなります
    *   タイマーが開始され、つま先立ちの姿勢を維持する必要があります
    *   QTZセンサーがかかとの位置をリアルタイムで監視を開始します
2.  監視メカニズム
    *   センサーには2つのボタン検出ポイントがあります
    *   かかとがいずれかのボタンを押し下げた瞬間 = **違反**
    *   システムは直ちに電気刺激による罰を発動します
    *   電気刺激は、2つのボタンの**両方が**押されていない状態になるまで停止しません
3.  罰システム
    *   🔴 即時罰 ：かかとがボタンを押し下げた瞬間に電気刺激が開始
    *   📈 進行強度 ：違反回数が増えるほど、電気刺激の強度が自動的に増加
    *   ⏱️ 持続罰 ：ボタンが押されている間は電気刺激が持続し、完全に上げられるまで続く
    *   🎮 二律背反罰：つま先立ちを一定時間続けた後、TD01デバイスを起動して干渉を開始
4.  ゲーム終了
    *   設定時間を堅持すると自動的にロックが解除されます
    *   システムは完全なパフォーマンスレポートを表示します
    *   違反回数、電気刺激回数などのデータを集計 💪 チャレンジのポイント
*   体力チャレンジ ：長時間のつま先立ちは脚の筋肉持久力を試します
*   意志力の試練 ：疲労に直面しても標準的な姿勢を維持する必要があります
*   精密制御 ：かかとのわずかな沈み込みも検出されます
*   心理戦 ：電気刺激の威嚇下で冷静さと集中力を保つ

### 🏆 勝利条件
設定時間の終了まで成功裏に堅持し、違反回数が少ないほど優秀な成績です：
*   🌟 パーフェクト ：0回違反
*   ⭐ 優秀 ：1～2回違反
*   👍 良好 ：3～5回違反
*   😅 要改善 ：5回以上違反

### ⚠️ 安全上の注意
*   心臓病などの疾患がなく、健康状態が良好であることを確認してください
*   低い強度と短い時間から始めることをお勧めします
*   ゲーム中に不快感を感じた場合は直ちに中止してください
*   18歳以上の成人のみが使用可能です
