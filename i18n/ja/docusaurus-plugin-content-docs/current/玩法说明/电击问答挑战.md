# 電気ショック・クイズチャレンジ

## ゲーム概要
クイズゲームは外部からの問題インポートをサポートする回答ゲームです。プレイヤーは制限時間内に問題に回答する必要があります。正解すると次の問題へ進み、不正解だと電気ショックの罰を受け、その問題は後ほど再回答するために末尾に回されます。

## ゲームの特徴
+ 📚 **カスタム問題集**: 外部JSON形式の問題ファイルのインポートをサポート
+ ⚡ **電気ショックの罰**: 問題に不正解すると電気ショックデバイスが作動し罰を与える
+ ⏰ **制限時間**: 各問題には回答時間の制限があり、時間切れは不正解とみなされる
+ 🎮 **多彩な操作方式**: マウスクリックとQTZキーデバイス操作をサポート
+ 🔒 **手動開始**: 手動開始モードを選択可能。自動ロックデバイスのボタンでゲームを起動

## 必要デバイス
### 必須デバイス
+ **電気ショックデバイス (DIANJI)**: 不正解時に電気ショックの罰を実行するために使用

### オプションデバイス
+ **自動ロックデバイス (ZIDONGSUO)**: ゲームの手動開始機能に使用
+ **QTZキーデバイス (QTZ)**: マウスの代わりに回答選択が可能

## ゲームパラメータ設定
### 問題設定
+ **問題ファイルパス**: 問題を含むJSONファイルのパスを指定
    - デフォルト: `<OUTTER_GAME>/QA-game/女仆行为规范考核题库.json`
    - `<OUTTER_GAME>` マーカーはパッケージ化後のoutter-gameフォルダを指すことをサポート

### 罰則設定
+ **電気ショック強度**: 不正解時の電気ショック強度 (10-100)
+ **電気ショック持続時間**: 電気ショックの持続時間 (1-10秒)

### 時間設定
+ **回答制限時間**: 各問題の回答時間制限 (10-300秒、0は無制限)

### 起動設定
+ **手動開始**: 有効化すると、自動ロックデバイスのボタンを押してゲームを開始する必要がある

## 問題ファイル形式
問題ファイルはJSON形式で、以下の構造を含む必要があります:

```json
{
  "questions": [
    {
      "question": "問題内容",
      "options": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
      "correct": 0,
      "explanation": "解答説明（任意）"
    }
  ]
}
```

### フィールド説明
+ `question`: 問題文
+ `options`: 選択肢の配列
+ `correct`: 正解のインデックス (0から開始)
+ `explanation`: 解答説明（任意）

## ゲームの流れ
1. **ゲーム準備**
    - 問題ファイルを読み込む
    - デバイスの接続状態を確認する
    - ゲームパラメータを初期化する
2. **ゲーム開始**
    - 手動開始モード: 自動ロックデバイスのボタン信号を待つ
    - 自動開始モード: すぐに最初の問題を開始する
3. **回答プロセス**
    - 問題と選択肢を表示する
    - 自動ロックデバイスをロックする (途中退出を防止)
    - カウントダウンを開始する
    - プレイヤーの回答選択を待つ
4. **回答判定**
    - **正解**: 正解表示をし、次の問題へ進む
    - **不正解または時間切れ**: 電気ショックの罰を実行し、問題をキュー末尾に戻して再回答する
5. **ゲーム終了**
    - 全ての問題に正解するとゲーム終了
    - 自動ロックデバイスをアンロックする
    - ゲーム統計情報を表示する

## 操作方式
### マウス操作
+ 選択肢ボタンをクリックして回答を選択

### QTZキー操作
+ **キー0**: 選択肢を上へ切り替え
+ **キー1**: 選択肢を下へ切り替え / 選択を確定
+ 選択中の選択肢はハイライト表示される

## 注意事項
1.  電気ショックデバイスが正しく接続・設定されていることを確認してください。電圧強度は自己調整してください。強度が高すぎることによる電気ショックの損害は自己責任となります。
2.  自動ロックはネットワーク条件などの要因で不安定な場合があります。脱出用の道具を必ず準備してください。
