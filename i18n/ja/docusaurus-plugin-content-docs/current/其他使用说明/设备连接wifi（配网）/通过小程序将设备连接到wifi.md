# ミニアプリでデバイスをWiFiに接続する方法

すべてのデバイスは、このドキュメントに従ってWiFiに接続できます。

**<font>注意：デフォルトWiFiを使用したい場合は、ネットワーク設定を行わないでください。デフォルトWiFi名は easysmart、パスワードは 11111111 です。デバイスは自動的にデフォルトWiFiに接続します。</font>**

ビデオチュートリアル:

抖音（TikTok）: [https://v.douyin.com/l_QWua94sRA/](https://v.douyin.com/l_QWua94sRA/)

YouTube: [https://www.youtube.com/watch?v=W7ITGIC0lw8](https://www.youtube.com/watch?v=W7ITGIC0lw8)

**<font>前提条件:</font>**

1.  環境に2.4G無線WiFiが存在すること。
2.  スマートフォンが現在その無線WiFiに接続されていること（2.4Gと5Gが統合されたSSIDの場合は、同じ名前の5G Wi-Fiに接続していても可能）。

成功しない場合は、[APPでデバイスをWiFiに接続する](./通过APP将设备连接到wifi.md)方法もお試しください。

## ステップ1: デバイスを起動する デバイスはライトが点灯しているはず
例:

![1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0.jpeg](./img/BwCUIIfLskyFx0zd/1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0-968887.jpeg)

## ステップ2: 微信（WeChat）でミニアプリ「物联地带蓝牙配网」を検索し、クリックして起動する
<font>注意：操作前にスマートフォンのBluetoothをオンにしてください</font>

起動後は以下のようになります。

![1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7.jpeg](./img/BwCUIIfLskyFx0zd/1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7-216457.jpeg)

「开始配网」をクリックします。数秒後に以下のようになります。

（デバイスが起動している必要があります。
また、デバイスがネットワーク設定に成功し、WiFiに接続した後は検索できません）

![1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa.jpeg](./img/BwCUIIfLskyFx0zd/1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa-204739.jpeg)

検索されたデバイスを選択します。

現在スマートフォンが接続しているWiFiのパスワードを入力します（必ず2.4G、または2.4Gと5Gが同名のもの）。

「下一步」をクリックします。

![1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9.jpeg](./img/BwCUIIfLskyFx0zd/1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9-470214.jpeg)

数秒後にネットワーク設定が成功します。

![1736778915718-1e769ec6-d814-49fe-91b6-771f19454784.jpeg](./img/BwCUIIfLskyFx0zd/1736778915718-1e769ec6-d814-49fe-91b6-771f19454784-492537.jpeg)

「配网失败」と表示された場合でも問題ありません。これも成功です。この時点でネットワーク設定は完了しています。

**<font>注意：デバイスがWiFiに接続すると自動的にBluetoothがオフになります。「失敗」と表示されても、再度検索してデバイスが見つからない場合は、すでに成功しています。</font>**
