# PCクライアントによるネットワーク設定（PCのBluetooth対応が必要）

<font>注意：この方法はまだテスト段階であり、不安定な可能性があります。失敗した場合は、ネットワーク設定を終了して再開するか、他の方法を試してください。</font>

**<font>注意：デフォルトWiFiを使用したい場合は、ネットワーク設定を行わないでください。デフォルトWiFi名はeasysmart、パスワードは11111111です。デバイスは自動的にデフォルトWiFiに接続します。</font>**

![1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4.png](./img/hURz820DTNcJYUCH/1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4-096925.png)

1. クライアントを開き、「デバイス管理」＞「デバイスネットワーク設定」をクリックします。

![1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed.png](./img/hURz820DTNcJYUCH/1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed-155975.png)

2. Wi-Fi名とWi-Fiパスワードを入力します（間違えないように注意してください。間違えると、ネットワーク設定が成功してもインターネットに接続できません）。
3. 「設定を変更」をクリックします。
4. 「ネットワーク設定を開始」をクリックすると、ネットワーク設定が開始されます。
