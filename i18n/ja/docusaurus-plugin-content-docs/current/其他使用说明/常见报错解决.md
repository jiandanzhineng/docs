# よくあるエラーと解決策

1. デバイスをネットワークに設定した後、クライアントを起動しても自動的にオンラインにならない
    1. ルーターが mDNS をサポートしているか確認してください。詳細は [ルーターが mDNS をサポートしているか確認する（ほとんどのルーターは対応）](./检测路由器是否支持mdns（大部分路由器都支持）.md) を参照してください。対応していない場合は以下の方法をお試しください：
        1. スマートフォンのホットスポットを使用してPCとデバイスを接続し、プレイする。
        2. ルーターと光回線終端装置（ONU）の接続関係を確認してください。ルーターのモードがAPモードの場合は、DHCPモードに変更してみてください。

2. vcruntime140.dll 関連のエラー

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

<font>Visual C++ Redistributable for Visual Studio 2015</font> をダウンロードしてインストールしてください。

3. 起動時に以下のエラーが発生する

<font>Unable to load emulator DLL</font>  
<font>(C:\beidanci\emqx\erts-13.2.2.4\bin\beam.smp.dll)</font>
