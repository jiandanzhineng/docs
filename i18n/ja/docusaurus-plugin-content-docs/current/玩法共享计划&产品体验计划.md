# 遊び方共有計画＆プロダクト体験計画

# 遊び方共有計画
ユーザーがデバイスを基に独自の遊び方を開発することを歓迎します

開発完了後、アップ主に連絡して対応する遊び方セットを上架することができます。販売後は、遊び方開発者に売上高の5％を還元します。

具体的には、Bilibiliアップ主、淘宝店主、またはQQグループの管理者にご連絡ください～



# プロダクト体験計画
1. 体験したい遊び方を選択する
2. プロダクト体験官申込みフォームを記入 [https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS](https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS)
3. 必要があればカスタマーサービスが連絡します
4. 商品を注文する
5. 使用動画を撮影して送信する
6. 商品代金をお支払いします～

いくつかの説明：プライバシー部位を露出する必要はありません
