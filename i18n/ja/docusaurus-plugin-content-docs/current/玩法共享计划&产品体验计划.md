# プレイスタイル共有プログラム＆製品体験プログラム

# プレイスタイル共有プログラム
ユーザーがデバイスを基に独自のプレイスタイルを開発することを歓迎します。

開発完了後は、アップ主に連絡して対応するプレイスタイルセットを掲載してもらい、販売後はプレイスタイル開発者に売上額の5％を還元します。

詳細は、Bilibiliのアップ主、淘宝店主、またはQQグループ管理者までご連絡ください～

# 製品体験プログラム
1. 体験したいプレイスタイルを選択
2. 製品体験官申込フォーム[https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS](https://rcnhcfc58y3x.feishu.cn/share/base/form/shrcnE8uStsXM6mJcSL2FPh25mS)を記入
3. 必要に応じてカスタマーサポートがご連絡します
4. 商品を注文
5. 使用動画を撮影して送信
6. 商品金額をお返しします～

備考：プライベート部分を露出する必要はありません
