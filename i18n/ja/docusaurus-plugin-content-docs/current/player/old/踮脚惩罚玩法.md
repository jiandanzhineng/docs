# つま先立ちペナルティプレイ

## 🎭 メイドのサボりペナルティゲーム v1.4.0
ハイテク体感自律トレーニングゲーム - つま先立ちチャレンジ

### 🎮 ゲームプレイ解説 📋 ゲーム準備
1.  デバイス設定：システムが自動的に3つのコアデバイスを接続します
    *   🔒 自動ロック：ゲーム開始時にロックされ、途中退出を防止します。[シンプルスマートロック端末](../device/简单智能锁定终端.md)
    *   ⚡ 電気刺激デバイス：即時のフィードバックペナルティを提供します。[シンプルスマートパルス（電気刺激）端末](../device/简单智能脉冲（电击）终端.md)
    *   📡 QTZ レーザーセンサー：踏み台を介してかかとが接地しているかどうかを監視します（かかとの下に配置）。[シンプルスマート距離検知端末](../device/简单智能距离感应终端.md)
+   **<font>こちらのセットをお買い上げ：</font>**[かかとを下ろすと電気刺激プレイセット](https://item.taobao.com/item.htm?id=945819055827)
+   パラメータ設定：ゲーム難易度をカスタマイズ可能
    *   ゲーム時間：1〜180分調整可能
    *   電気刺激強度：10〜100レベル
    *   進行モード：ルール違反回数が増えるほど、ペナルティが重くなります。 🎯 具体的なプレイ方法

コアルール ：つま先立ち姿勢を保持し、かかとが絶対に地面に触れてはいけません！

1.  ゲーム開始
    *   システムが自動ロックされ、途中退出できません。
    *   カウントダウン開始、つま先立ち姿勢を維持する必要があります。
    *   QTZセンサーがかかとの位置をリアルタイムで監視開始。
2.  監視メカニズム
    *   センサーには2つのボタン検知点があります。
    *   かかとがいずれかのボタンを押し下げた瞬間 = ルール違反。
    *   システムは直ちに電気刺激ペナルティを発動します。
    *   両方のボタンが押されていない状態になった時のみ、電気刺激が停止します。
3.  ペナルティシステム
    *   🔴 即時ペナルティ ：かかとがボタンを押し下げた瞬間に電気刺激開始。
    *   📈 強度進行 ：ルール違反回数が増えるほど、電気刺激強度が自動的に増加。
    *   ⏱️ 持続ペナルティ ：ボタンが押されている間は電気刺激が継続し、完全に離れるまで止まりません。
    *   🎮 ジレンマペナルティ：つま先立ちを一定時間続けた後、TD01デバイスを起動して干渉します。
4.  ゲーム終了
    *   設定時間まで持ちこたえると自動的にロック解除されます。
    *   システムが完全なパフォーマンスレポートを表示します。
    *   ルール違反回数、電気刺激回数などのデータを集計。 💪 挑戦のポイント
+   体力チャレンジ ：長時間のつま先立ちは脚の筋持久力を試します。
+   意志力の試練 ：疲労に直面しても標準姿勢を維持する必要があります。
+   精密コントロール ：かかとのわずかな沈み込みも検知されます。
+   心理的駆け引き ：電気刺激の威圧下で冷静さと集中力を保ちます。 🏆 勝利条件
設定時間終了まで持ちこたえることに成功し、ルール違反回数が少ないほど優秀なパフォーマンスです：
+   🌟 パーフェクトパフォーマンス ：0回違反
+   ⭐ 優れたパフォーマンス ：1〜2回違反
+   👍 良好なパフォーマンス ：3〜5回違反
+   😅 改善の余地あり ：5回以上の違反

### ⚠️ 安全上の注意
+   心臓病などの疾患がないなど、健康状態を確認してください。
+   低強度・短時間から始めることをお勧めします。
+   ゲーム中に気分が悪くなった場合は直ちに中止してください。
+   18歳以上の成人のみご使用ください。
