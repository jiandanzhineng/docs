# 寸止めプレイアップグレード版（3段階版）

## はじめに
これは気圧センサーに基づく寸止めトレーニングプレイです。システムは気圧値に応じて振動強度をインテリジェントに調節し、ユーザーが興奮の瀬戸際を維持するよう導きます。気圧が臨界値を超えた場合、電気ショック罰則がトリガーされます。

## ゲームフロー
ゲームは以下の状態サイクルを含みます：

![フロー図](../img/cunzhi3phase.jpg)

1.  **平静期 (Initial/Sub Calm)**
   -   振動強度がゆっくり上昇し、ランダムな擾乱が伴います。
   -   **目標**：興奮度を高め、気圧を上げます。
   -   **遷移**：気圧が「中間圧力」を超えると中期に移行。

2.  **中期 (Middle)**
   -   インテリジェント制御：気圧が高いほど振動強度が低く（早く爆発するのを防ぐ）、気圧が低いほど振動強度が高く（興奮を維持）なりま す。
   -   **遷移**：
      -   気圧 >= 「臨界気圧」：**電気ショック**がトリガーされ、ギリギリ期に入ります。
      -   気圧 < 「中間圧力」：平静期に戻ります。

3.  **ギリギリ期 (Edging)**
   -   **電気ショック発動！** 振動停止。
   -   **遷移**：気圧が臨界値以下に下がった後、冷却遅延期に入ります。

4.  **遅延期 (Delay)**
   -   しばらくの間（冷静遅延）待機します。
   -   **遷移**：遅延終了後、現在の気圧に応じて中期（中間圧力より大きい時）または平静期（中間圧力より小さい時）に戻ります。

## 必要デバイス
1.  **気圧センサー (QIYA)**：必須。興奮度を監視します。
2.  **偏心モーターコントローラー (TD01)**：必須。振動刺激を提供します。
3.  **電気ショックデバイス (DIANJI)**：オプション。ギリギリ時の罰則に使用します。
4.  **自動ロック (ZIDONGSUO)**：オプション。ゲーム開始時に自動ロック、終了時にロック解除。

## 使用説明 {#use_instruction}

1.  気圧デバイスの組み立てを完了します。
   -  デバイス組み立てビデオチュートリアル：https://youtu.be/QQaT823gKHc
   -   夸克：https://pan.quark.cn/s/03b8d2dab2ff
   -   OSR6バージョンは [OSR6プレイ説明](../../device/osr6-control-plugin.md) をご参照ください。
2.  すべてのデバイスをネットワークに接続します。
   -   スマートフォン：[スマホクライアント](../client/new-phone-client.md)
   -   PC：[PC版コントロールクライアント](../client/PC版控制客户端.md)
3.  気圧デバイスを体内に挿入し、通常20-25kPaまで空気を注入します。圧力を軽く押した時に気圧が明確に変化するようにし、かつ適度に低い圧力で維持することで空気漏れを遅らせます。
注意：気圧センサー起動後の最初の3分間は気圧が不安定になる（持続的に低下または上昇する）ことがあります。これはセンサーのウォームアップ中です。この状況に遭遇した場合は、ゲーム開始前にしばらくお待ちください。
4.  中間気圧と臨界気圧を設定します。初めてプレイする際は、プレイしながら調整できます。この気圧で我慢できなくなったら少し下げ、刺激が足りなければ少し上げて調整してください。

## 設定説明

| パラメーター名 | 説明 | 推奨値 |
| :--- | :--- | :--- |
| **ゲーム時間** | ゲームの総実行時間（分）。 | 20 |
| **臨界気圧** | 電気ショックおよび寸止めをトリガーする最大気圧しきい値 (kPa)。 | 23 |
| **TD01最大強度** | 振動の最大強度 (1-255)。 | 255 |
| **中間(興奮)圧力** | インテリジェント制御段階に入る圧力閾値 (kPa)。 | 22 |
| **電気ショック強度** | 罰則時の電圧強度 (V)。 | 20 |
| **電気ショック持続** | 罰則の持続時間 (秒)。 | 3 |
| **低圧遅延** | 寸止めトリガー後の冷却・休憩時間 (秒)。 | 5 |
| **強度増加速度上限** | 振動が強まる最速速度。急激な変化を防ぎます。 | 10 |
| **強度ランダム擾乱** | 振動強度のランダムな変動幅 (%)。 | 0 |
| **遅延後段階的向上** | 平静期に振動が毎秒自動的に増加する値。 | 10 |

## 安全上のご注意

1.  電気パルスデバイスの電圧は高く設定しすぎないでください。電流を胴体に流さないでください。
