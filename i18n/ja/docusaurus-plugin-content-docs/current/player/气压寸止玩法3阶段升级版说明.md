# 寸止めプレイアップグレード版（3フェーズ版）第2世代

## はじめに
これは気圧センサーに基づいた寸止めトレーニングプレイです。システムは常時気圧値を監視し、振動強度を知的に調整して、ユーザーが興奮の淵に留まるよう誘導します。気圧が限界値を超えた場合、電気ショックによるペナルティが発生します。

## バージョンアップグレード説明
第2世代バージョンでは、電気刺激、気圧、振動の3つのデバイスを統合・アップグレードし、ひとつのデバイスにまとめました。設定方法に変化はありません。

旧世代デバイスをご利用の場合は、チュートリアル [気圧寸止めプレイ3フェーズアップグレード版説明](./old/气压寸止玩法3阶段升级版说明.md) をご参照ください。

## ゲームの流れ
ゲームは以下の状態が循環します:

![フロー図](./img/cunzhi3phase.jpg)

1.  **平穏期 (Initial/Sub Calm)**
    *   振動強度がゆるやかに上昇し、ランダムな揺らぎが伴います。
    *   **目標**: 興奮度を上げ、気圧を増加させます。
    *   **遷移**: 気圧が「中間圧力」を超えると中期に移行します。

2.  **中期 (Middle)**
    *   知的制御: 気圧が高いほど振動強度は低く（早すぎる爆発を防ぐ）、気圧が低いほど振動強度は高くなります（興奮を維持）。
    *   **遷移**:
        *   気圧 >= 「臨界気圧」: **電気ショック**が発生し、寸止め期に移行します。
        *   気圧 < 「中間圧力」: 平穏期に戻ります。

3.  **寸止め期 (Edging)**
    *   **電気ショック発生！** 振動は停止します。
    *   **遷移**: 気圧が臨界値を下回った後、遅延期に移行します。

4.  **遅延期 (Delay)**
    *   一定時間待機します（冷静遅延）。
    *   **遷移**: 遅延終了後、現在の気圧に応じて中期（中間圧力以上）または平穏期（中間圧力未満）に戻ります。

## 必要デバイス
1.  **寸止めセンサー (CUNZHI01)**: 必須。興奮度を監視し、振動刺激を提供、および寸止めのペナルティ（電気ショック）を行います。
2.  **オートロック (ZIDONGSUO)**: 任意。ゲーム開始時に自動でロックされ、終了時に解除されます。


## 使用説明 {#use_instruction}

1.  デバイスの組み立てを完了し、装着します。（詳しくは[操作手順](./寸止玩法二代使用教程.md#operation-steps)のステップ2をご参照ください）
    *   OSR6版の方は [OSR6プレイ説明](../device/osr6-control-plugin.md) をご参照ください。
2.  すべてのデバイスをネットワークに接続します。
    *   スマートフォン: [スマートフォンクライアント](/docs/player/new-phone-client) をご参照ください。
    *   パソコン: [PC版コントロールクライアント](./client/PC版控制客户端.md) をご参照ください。

3.  中間圧力と臨界気圧を設定します。初めてプレイする際は、プレイしながら調整できます。我慢できない圧力なら下げ、感じにくければ上げて調整してください。

## 設定説明

| パラメータ名 | 説明 | 推奨値 |
| :--- | :--- | :--- |
| **ゲーム時間** | ゲームの総実行時間（分）。 | 20 |
| **臨界気圧** | 電気ショックと寸止めが発生する最高気圧しきい値 (kPa)。 | 23 |
| **TD01最大強度** | 振動の最大強度 (1-255)。 | 255 |
| **中間(興奮)圧力** | 知的制御段階に入るための圧力しきい値 (kPa)。 | 22 |
| **電気ショック強度** | ペナルティ時の電圧強度 (V)。 | 20 |
| **電気ショック持続時間** | ペナルティの持続時間（秒）。 | 3 |
| **低圧遅延** | 寸止め発生後のクールダウン休憩時間（秒）。 | 5 |
| **強度増加率上限** | 振動強度が増加する最大速度。急激な変化を防ぎます。 | 10 |
| **強度ランダム揺らぎ** | 振動強度のランダムな変動幅 (%)。 | 0 |
| **遅延後徐々に上昇** | 平穏期に振動強度が毎秒自動的に増加する値。 | 10 |

## 安全使用説明

1.  電気パルスデバイスの電圧は高く設定しすぎず、電流が胴体を通らないように注意してください。
