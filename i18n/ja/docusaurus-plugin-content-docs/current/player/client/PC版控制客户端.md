# PCクライアントの使用ガイド

1. ダウンロードリンク：[https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip](https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip)
2. **解凍してからインストールしてください （解凍に失敗した場合は [7zip](https://www.7-zip.org/download.html) をインストール後、そのソフトで解凍してください）**
3. **クライアントを起動する**
4. **PCのWiFiホットスポットを有効にし、WiFi名を `easysmart`、パスワードを `11111111`（8つの1）、バンドを**2.4G**に設定する（[PCホットスポットなしの設定方法](#无电脑热点情况使用方式)）**
注意：デバイスが既に5Gの他のWiFiに接続している場合、デバイスが2.4Gで設定されていても5Gバンドのままになる可能性があります。その場合は、デバイスを2.4G WiFi（または有線LAN）に接続する必要があります。
5. **デバイスがWiFiに接続しているのにクライアントに表示されない場合は、ファイアウォールにブロックされている可能性があります。PCのファイアウォールを無効にしてください。**
6. **デバイスを起動し、起動後約15〜30秒で自動的にPCに接続されます**
7. **これで楽しく操作できます**

## PCホットスポットがない場合の使用方法
1. デバイスをWiFiに接続する：[APPでデバイスをWiFiに接続する](../other/设备连接wifi（配网）/通过APP将设备连接到wifi.md)を参考に、デバイスを自宅のWiFiに接続します（必ず2.4G WiFiを選択してください）
2. クライアントでmDNSを有効にする
