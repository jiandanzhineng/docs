# つま先立ちペナルティ遊び v2

High-tech体感自律トレーニングゲーム - つま先立ちチャレンジ

### v1.4.0 アップデート内容:
1.  **TD01デバイス追加**: 長時間の退屈を防ぐため、一定時間ショックが発生しないと起動し、強度が段階的に増加します
2.  **Zidongsuoキー操作でのゲーム開始を追加**: 一人での操作しやすくするため、マニュアルスタートモード有効時はキー操作でゲームを開始できます
3.  **Zidongsuo、TD01はオプションデバイス**: この二つのデバイスが無くても通常通りゲームを遊べます。ショックデバイスとQTZセンサーのみで動作可能です

## 🎮 ゲームプレイ詳細
### 📋 ゲーム準備
#### 1. デバイス設定
システムは自動的に4つの核心デバイスを接続します：

+ 🔒 **自動ロック**: ゲーム開始時にロックし、途中退出を防止（オプションデバイス）
+ ⚡ **ショックデバイス**: 即時フィードバックのペナルティを提供
+ 📡 **QTZレーザーセンサー**: 二つのボタンでかかとが地面に触れているか監視（かかとの下に設置）
+ 🎮 **TD01デバイス**: オプションデバイス。一定時間ショックが発生しないと起動し、強度を徐々に増加させてつま先立ちを妨害します。

#### 2. パラメータ設定
ゲームの難易度をカスタマイズ可能：

+ **ゲーム時間**: 1-360分で調整可能
+ **ショック強度**: 10-100レベル
+ **プログレッシブモード**: 違反回数が多いほど、ペナルティが重くなります
+ **安全制限**: 30Vの安全制限を超えることを許可するか選択可能
+ **マニュアルスタート**: auto_lockデバイスのキークリックを待ってからゲームを開始するかを選択可能
+ **TD01遅延時間**: 1-60秒で調整可能。ショックが発生しないまま何秒後にTD01を起動するか
+ **TD01強度増加量**: 5秒毎に増加する強度値（1-100）

## 🎯 具体的なプレイ方法
### 基本ルール
つま先立ちの姿勢を保ち、かかとを絶対に地面に触れてはいけません！

#### 1. ゲーム開始時
+ システムが自動的にロック（自動ロックデバイスがある場合）、途中で退出できなくなります
+ マニュアルスタートモードを有効にした場合、auto_lockデバイスのキーをクリックしてタイマーを開始する必要があります
+ タイマー開始後、つま先立ちの姿勢を維持する必要があります
+ QTZセンサーがかかとの位置のリアルタイム監視を開始します

#### 2. 監視メカニズム
+ センサーは二つのボタン検出ポイント（button0とbutton1）を持っています
+ かかとがいずれかのボタンを押し下げると = 違反
+ システムは直ちにショックペナルティを発生させます
+ 両方のボタンが押されていない場合にのみ、ショックは停止します

#### 3. ペナルティシステム
+ 🔴 **即時ペナルティ**: かかとがボタンを押し下げた瞬間にショックが開始します
+ 📈 **プログレッシブ強度**: 違反回数が多いほど、ショック強度が自動的に増加します
+ ⏱️ **持続ペナルティ**: ボタンが押されている間はずっとショックが継続し、完全に離すまで続きます
+ 🎮 **TD01強化ペナルティ**: 指定時間ショックが発生しない状態が続くとTD01デバイスが起動し、強度が5秒毎に増加します

#### 4. TD01デバイスメカニズム
+ **遅延起動**: 指定秒数ショックが発生しないと自動起動します
+ **強度増加**: 5秒毎に設定された強度値を自動的に増加させます
+ **最大強度**: 0-255の範囲をサポートし、安全制限の影響を受けません
+ **自動停止**: ショックが発生すると自動的に停止し、リセットされます

#### 5. ゲーム終了時
+ 設定時間を耐え抜くと自動的にロック解除されます
+ システムが完全なプレイ報告を表示します
+ 違反回数、ショック回数などのデータを統計します

## 💪 チャレンジポイント
+ **体力チャレンジ**: 長時間のつま先立ちで脚の筋肉の持久力を試されます
+ **精神力の試練**: 疲労に直面しても正しい姿勢を維持する必要があります
+ **精密制御**: ほんのわずかなかかとの沈み込みも検知されます
+ **心理ゲーム**: ショックの威嚇下で冷静さと集中力を保ちます
+ **二重のプレッシャー**: ショックを避けつつ、TD01デバイスの妨害にも対処しなければなりません

## 🏆 勝利条件
設定時間終了まで成功させ、違反回数が少ないほど優秀なパフォーマンス：

+ 🌟 **完璧なプレイ**: 違反0回
+ ⭐ **優秀なプレイ**: 違反1-2回
+ 👍 **良好なプレイ**: 違反3-5回
+ 😅 **改善の余地あり**: 違反5回以上

## ⚠️ 安全上の注意
+ 心臓病、てんかん等の疾患がない、健康な状態であることを確認してください
+ 低強度・短時間から始めることを推奨します
+ ゲーム中に体調不良を感じた場合は直ちに中止してください
+ 18歳以上の成人のみ使用可能です
+ ショック強度は慎重に設定してください。高すぎると電撃傷害を引き起こす可能性があり、リスクは自己責任で負ってください
+ 安全制限を超える機能を有効にする場合のリスクは自己負担となります
+ ゲーム環境の安全を確保し、監督者がいることを推奨します

## 🔧 特長機能
### マニュアルスタートモード
+ ゲームロード完了後、待機画面が表示されます
+ auto_lockデバイスがまずロックされます
+ auto_lockデバイスのキーをクリックしてゲームタイマーを開始する必要があります
+ 準備時間が必要なシチュエーションに適しています

### リアルタイムショックメカニズム
+ いずれかのボタンが押された瞬間に持続的なショックが開始します
+ 全てのボタンが離された瞬間にショックが停止します
+ 最も直接的なフィードバックメカニズムを提供します

### TD01強化システム
+ 追加のチャレンジを提供するオプションデバイス
+ 良いプレイをしているときに難易度を増加させます
+ 強度が徐々に増加し、ゲームの緊張感を高めます
+ 独立した強度制御をサポートしています
