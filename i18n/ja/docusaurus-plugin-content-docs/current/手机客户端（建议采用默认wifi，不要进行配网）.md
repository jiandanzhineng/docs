対応機種：**<font>Androidスマートフォンのみ</font>**および2025年11月1日以降に出荷された製品。以前のデバイスは、

[ハードウェアファームウェア書き込み説明](./其他使用说明/硬件固件烧录说明.md)

に従ってファームウェアを更新可能

# 使用方法
1. termuxをダウンロード ダウンロードリンク：[https://update.ezsapi.top/termux.apk](https://update.ezsapi.top/termux.apk)
2. termuxをインストールし、termuxの電源管理を**<font>制限なし、バックグラウンド固定</font>**に設定（バックグラウンド終了防止）
3. termuxを起動

**<font>初回実行時は以下のコマンドをコピーし、termuxに貼り付けてEnterを押してください</font>**

```bash
curl -fsSL https://update.ezsapi.top/run_control_panel.sh | sh
```

**<font>初回以外</font>**の場合、以下のコマンドで起動速度を向上可能（更新したい場合は上記コマンドを実行）

```bash
cd "${HOME}/control-panel-stable" && npm run dev:all
```

実行完了まで待機 下図のような画面が表示されます（20分経っても表示されない場合はよくある質問1を参照）

![1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663.jpeg](./img/aeI6B3lspG60kJMv/1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663-159999.jpeg)

4. スマートフォンのホットスポットを有効化 ホットスポット名は「easysmart」、パスワードは「11111111」に設定することを推奨（デバイスは自動接続）。異なる場合はその他の使用方法のデバイス配網手順を参照し、デフォルト設定を推奨
5. スマートフォンのブラウザで 127.0.0.1:5173 にアクセス
6. デバイス管理にデバイスが接続されていることを確認 ゲームリストからゲームを起動可能（空の場合はページを更新）

# よくある質問
## 正常に起動しない
ネットワーク問題により起動失敗の可能性あり。以下の解決方法を試してください

1. termuxを再インストール（またはtermuxの全データを削除）し、再度実行
2. スマートフォンを再起動し、termuxを再インストール（またはtermuxの全データを削除）、再度実行
3. ダウンロード速度が遅い場合は、夜7時～12時の時間帯を避けてダウンロード（この時間帯はネットワークが混雑）。午前中は通常快適。

また、termuxの<font>バックグラウンド制限解除</font>設定を確認してください

上記方法でも解決しない場合は、QQグループ 970326066 に参加し、管理者にご連絡ください

## デバイスがスマートフォンのホットスポットに接続できない
スマートフォンのホットスポットを2.4GHzに切り替えてください。それでも接続できない場合は、他のデバイスで同じホットスポットに接続を試みてください

可能性1：他のスマートフォンもホットスポットに接続できない場合、スマートフォンのホットスポットに問題がある可能性
可能性2：ホットスポットが2.4GHzに切り替わらない問題。あるユーザーはルーターの周波数帯域を2.4GHzと5GHzに分離することで解決（原因不明）

可能性3：ユーザーが配網操作を実行し、デバイスがデフォルトWi-Fiに接続しなくなった場合。配網アプリを使用してデバイスをデフォルトのWi-Fi名とパスワードに再接続してください（アプリでの配網Wi-Fi名は変更可能）
