# 使用方法
1. Termuxをダウンロードします。ダウンロードURL：[https://update.ezsapi.top/termux.apk](https://update.ezsapi.top/termux.apk)
2. Termuxをインストールし、Termuxの電源管理設定を**<font>制限なし、バックグラウンドロック</font>**に設定して、バックグラウンドでの強制終了を防ぎます。
3. Termuxを開きます。

**<font>初回実行時は、以下のコマンドをコピーしてTermuxに貼り付け、Enterキーを押してください</font>**

```bash
curl -fsSL https://update.ezsapi.top/run_control_panel.sh | sh
```

**<font>初回実行でない場合</font>**は、以下のコマンドを実行すると読み込み速度が向上します（更新したい場合は上記のコマンドを使用してください）。

```bash
cd "${HOME}/control-panel-stable" && npm run dev:all
```

実行が完了するまで待ちます。以下の画像のような表示になります（20分経っても類似の画像が表示されない場合は、よくある問題1を参照してください）。

<img src={require('./img/aeI6B3lspG60kJMv/1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663-159999.jpeg').default} width="30%" height="auto" alt="1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663.jpeg" />

4. スマートフォンのホットスポットをオンにします。ホットスポット名を「easysmart」、パスワードを「11111111」（8つの1）に設定することを推奨します。デバイスは自動的に接続します。これ以外の設定の場合は、その他の使用説明にあるデバイスのネットワーク設定方法を参照して設定してください。デフォルト設定の使用を推奨します。
5. スマートフォンのブラウザで 127.0.0.1:5173 にアクセスします。
6. これで、デバイス管理にデバイスが接続されているのが確認でき、ゲームリストからゲームを起動できるはずです（空の場合はページを更新してみてください）。

# よくある問題
## 正常に起動しない
ネットワークの問題により、起動に失敗する可能性があります。この場合、2つの解決策があります。

1. Termuxを再インストールする（またはTermuxの全データを消去する）、その後再実行する。
2. スマートフォンを再起動し、Termuxを再インストールする（またはTermuxの全データを消去する）。その後、再度実行する。
3. ダウンロード速度が遅すぎる場合は、夜7時から12時の時間帯を避けてダウンロードしてください。この時間帯は通常ネットワークが混雑します。午前中は一般的に混雑が少ないです。

また、Termuxの**<font>バックグラウンド制限なし</font>**が設定されているか確認してください。

上記の方法を試しても解決しない場合は、QQグループ 970326066 に参加してグループオーナーにご連絡ください。

## デバイスがスマートフォンのホットスポットに接続できない
スマートフォンのホットスポットを2.4GHzに切り替えてください。それでもダメな場合は、他のデバイスでこのスマートフォンのホットスポットに接続してみてください。
考えられる原因1：他のスマートフォンでもこのホットスポットに接続できない場合があり、スマートフォンのホットスポットに問題があることを示しています。
考えられる原因2：スマートフォンのホットスポットの問題である可能性もあります。原因はホットスポットが2.4GHzに切り替わらないままであることのようです。あるユーザーは、ルーターの帯域集約を2.4GHzと5GHzに分けることで解決しましたが、解決理由は不明です。

考えられる原因3：ユーザーがネットワーク設定操作を行い、デバイスがデフォルトのWi-Fiに接続しなくなった場合です。この場合は、ネットワーク設定アプリを使用して、デバイスをデフォルトのWi-Fi名とパスワードに再接続してください。アプリの設定Wi-Fi名は変更可能です。
