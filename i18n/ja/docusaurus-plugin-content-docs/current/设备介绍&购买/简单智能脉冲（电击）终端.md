注意事項:  
1**. デバイスはデフォルトでセーフモードになっており、このときの電圧は36V以下です。電圧を上げたい場合は、デバイスのsafe属性を0に設定してください。このモードでは電圧の最大制限は80Vになりますが、このモードでは各自で安全に注意し、問題が発生した場合は自己責任で対処してください。**

2. **出力を得るには、必ずコネクタを奥までしっかりと差し込んでください。**
