# シンプル・インテリジェント・ロック端末

![1750851448187-f17fa36f-e863-4896-9329-9b00b6095a9b.png](./img/ZGvq049xBaouj3MU/1750851448187-f17fa36f-e863-4896-9329-9b00b6095a9b-531187.png)

購入リンク： [https://item.taobao.com/item.htm?id=925058360048](https://item.taobao.com/item.htm?id=925058360048)

機能概要：

Wi-Fi接続、無線制御による端末のロック/アンロック、Type-C充電に対応

ボタンを60秒間長押しで強制アンロックが可能

電圧が20%以下になると自動的にアンロック

スイッチの方向：充電ポートから遠ざける方向がON、それ以外はOFF

注意：

1.  使用時には脱出用の道具を必ず準備してください！ネットワーク等の問題により、デバイスは確実にアンロックできることを保証しません！
