# 既知の問題

1. 一部のルーターではデバイスのネットワーク設定が成功しない場合がありますが、その際はスマートフォンのWi-Fiをアクセスポイントとして使用できます。
2. デバイスを初めて受け取ったときに、ネットワーク設定が失敗したり、設定後にオンラインにならなかったりする原因は、電池が切れている可能性があります。充電するか電源を接続し、再起動してみてください。
3. MDNSを起動する前に、必ずネットワーク環境が安定していることを確認してください。そうしないと、MDNSドメインのバインドに異常が発生することがあります（例えば、スマートフォンのホットスポット環境では、まずホットスポットに接続してからMDNSを起動する必要があります）。
4. スマートフォンをデバイスに接続した後、ネットワーク設定ボタンがグレー表示される場合があります。その際は、スマートフォンを再起動してみてください。
5. ホットスポット接続モードでは、ネットワークカードの品質によって、一定時間ごとに短い動作の遅延が発生する可能性があります。
