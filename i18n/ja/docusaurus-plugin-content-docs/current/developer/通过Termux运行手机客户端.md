# 使用手順
1. Termuxをダウンロード ダウンロード先：[https://update.ezsapi.top/termux.apk](https://update.ezsapi.top/termux.apk)
2. Termuxをインストールし、Termuxの電源管理設定を<font>制限なし、バックグラウンドロック</font>に設定し、バックグラウンドでの停止を防ぐ
3. Termuxを起動

**<font>初回実行時は、以下のコマンドをコピーしてTermuxに貼り付け、Enterキーを押してください</font>**

```bash
curl -fsSL https://firmware.undersilicon.cn/control-panel/run_control_panel_stable.sh | sh
```

**<font>初回実行でない場合</font>**は、以下のコマンドを実行すると読み込み速度が向上します（更新したい場合は上記のコマンドを使用してください）

```bash
cd "${HOME}/control-panel-stable" && npm run dev:all
```

実行完了を待機 以下のような画面になるはずです（20分経っても類似画像が表示されない場合は、よくある質問1を参照してください）

<img src={require('./img/aeI6B3lspG60kJMv/1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663-159999.jpeg').default} width="30%" height="auto" alt="1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663.jpeg" />

4. モバイルホットスポットを起動 ホットスポット名を「easysmart」、パスワードを「8個の1」、周波数を「2.4GHz」に設定することをお勧めします。デバイスが自動的に接続します（これらの設定でない場合は、他の使用方法のデバイスネットワーク設定方法を参照してネットワーク設定を行ってください。デフォルト設定の使用をお勧めします）
5. モバイルブラウザで 127.0.0.1:5173 にアクセス
6. デバイス管理でご利用のデバイスが接続済みであることを確認し、ゲーム一覧からゲームを起動できます（空の場合はページを更新してください）

# よくある質問
## 正常に起動しない
ネットワークの問題により、起動が失敗する可能性があります。解決策は2つあります。

1. Termuxを再インストールする（またはTermuxの全データを消去した後）、再度実行する
2. スマートフォンを再起動し、Termuxを再インストールする（またはTermuxの全データを消去した後）、再度実行する
3. ダウンロード速度が遅すぎる場合は、19:00から24:00までのネットワークが一般的に混雑している時間帯を避けてダウンロードを行ってください。午前中は通常スムーズです。

また、Termuxが<font>バックグラウンド無制限</font>に設定されているか確認してください。

上記の2つの方法を試しても解決しない場合は、QQグループ 970326066 に参加してグループ管理者にご連絡ください。

## デバイスがモバイルホットスポットに接続できない
モバイルホットスポットを2.4GHzに切り替えてください。それでも接続できない場合は、他のデバイスでこのスマートフォンのホットスポットに接続してみてください。
原因1：他のスマートフォンでもモバイルホットスポットに接続できないことがあった場合、スマートフォンのホットスポットに問題がある可能性があります。

原因2：モバイルホットスポットの問題。一部のスマートフォンは5GHz Wi-Fiに接続しているときに2.4GHzホットスポットを設定できない場合があります。この場合は、スマートフォンのWi-Fi接続を切断し、モバイルデータ通信のみを使用してインターネットに接続した後、Wi-Fiホットスポットを起動してください。

原因3：ユーザーがネットワーク設定操作を行い、デバイスがデフォルトWi-Fiに接続しなくなった場合。この場合は、設定用アプリを使用してデバイスをデフォルトのWi-Fi名とパスワードに再接続してください。アプリでの設定用Wi-Fi名は変更可能です。

## Termuxが「Unable to install bootstrap」と表示する

Termuxをスマートフォンの分身機能やアプリの分身モードにインストールしないでください。

## モバイルブラウザで 127.0.0.1:5173 にアクセスできない

Termuxの電源管理を制限なしに設定し、かつバックグラウンドロックを設定してバックグラウンドでの停止を防いでください。
