# Known Issues

1. Device provisioning may fail under some routers. In such cases, you can try using a mobile hotspot.
2. If a new device fails to provision or does not come online after provisioning, it may be due to low battery. Try charging it or keeping it plugged in, then power cycle the device.
3. Ensure that the network condition is stable before enabling mDNS; otherwise, it may cause abnormal mDNS domain binding (e.g., when using a mobile hotspot, connect to the hotspot first, then enable mDNS).
4. If the provisioning button is grayed out after connecting the device to a phone, try restarting the phone.
5. In hotspot connection mode, action delays may occur intermittently due to limitations of the network card quality.
