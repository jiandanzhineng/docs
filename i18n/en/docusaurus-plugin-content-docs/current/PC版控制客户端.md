# New PC Client User Guide

1. Download [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2. **<font>Please extract and then install. If extraction fails, install</font>**[7zip](https://www.7-zip.org/download.html)**<font> and use that software to extract.</font>**
3. **<font>Start the client</font>**
4. **<font>Enable PC hotspot, set WiFi name to easysmart, password to eight 1s, and band to</font>****<font>2.4G</font>****<font> band (see below if no PC hotspot)</font>**
5. **<font>(Important)</font>****<font>If the device connects to WiFi but doesn't appear in the client, it's being blocked. Turn off the PC firewall.</font>**
6. **<font>Start the device. It will automatically connect to the PC about 15-30 seconds after startup.</font>**
7. **<font>You can start having fun!</font>**

 

## Usage Without PC Hotspot
1. Configure device network: Refer to [Connecting Device to WiFi via APP](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) to connect the device to your home WiFi (must be 2.4G WiFi).
2. Enable mDNS in the client.
