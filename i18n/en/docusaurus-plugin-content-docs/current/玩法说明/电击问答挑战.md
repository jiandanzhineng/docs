# Electric Shock Quiz Challenge

## Game Introduction
The quiz game is a question-and-answer gameplay that supports importing external question sets. Players must answer questions within a set time limit. Correct answers allow progression to the next question, while incorrect answers trigger an electric shock punishment and the question is moved to the back of the queue to be answered again.

### Punishment Settings
+ **Shock Intensity**: Electric shock intensity for incorrect answers (10-100)
+ **Shock Duration**: Duration of the electric shock (1-10 seconds)

### Time Settings
+ **Answer Time Limit**: Time limit for answering each question (10-300 seconds, 0 for no limit)

### Start Settings
+ **Manual Start**: When enabled, the game can only be started by pressing the auto-lock device button

## Question File Format
The question file should be in JSON format with the following structure:

```json
{
  "questions": [
    {
      "question": "Question content",
      "options": ["Option A", "Option B", "Option C", "Option D"],
      "correct": 0,
      "explanation": "Answer explanation (optional)"
    }
  ]
}
```

### Field Descriptions
+ `question`: The question text
+ `options`: Array of answer options
+ `correct`: Index of the correct answer (starting from 0)
+ `explanation`: Explanation of the answer (optional)

## Game Flow
1. **Game Preparation**
    - Load the question file
    - Check device connection status
    - Initialize game parameters
2. **Game Start**
    - Manual start mode: Wait for the auto-lock device button signal
    - Auto-start mode: Immediately begin with the first question
3. **Answering Process**
    - Display the question and options
    - Auto-lock device locks (prevents mid-game exit)
    - Start the countdown timer
    - Wait for the player to select an answer
4. **Answer Judgment**
    - **Correct**: Display correct prompt, proceed to the next question
    - **Incorrect or Timeout**: Trigger electric shock punishment, move the question to the end of the queue to be answered again
5. **Game End**
    - Game ends after all questions are answered correctly
    - Auto-lock device unlocks
    - Display game statistics

## Control Methods
### Mouse Control
+ Click the option buttons to select an answer

### QTZ Button Control
+ **Button 0**: Move selection up
+ **Button 1**: Move selection down / Confirm selection
+ The selected option will be highlighted

2. The auto-lock device may be unstable due to network conditions and other factors. Be sure to prepare tools for emergency release.
