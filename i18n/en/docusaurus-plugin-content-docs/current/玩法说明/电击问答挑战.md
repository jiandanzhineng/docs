# Electric Shock Quiz Challenge

## Game Introduction
The quiz game is a question-answering gameplay that supports importing external question sets. Players must answer questions within a specified time limit. Answering correctly advances to the next question, while an incorrect answer triggers an electric shock punishment, and the question is moved to the back of the queue to be answered again.

## Game Parameter Configuration
### Question Settings
+ **Question File Path**: Specifies the path to the JSON file containing questions
    - Default: `<OUTTER_GAME>/QA-game/Maid Behavior Standard Exam Question Bank.json`
    - Supports the `<OUTTER_GAME>` tag pointing to the packed outter-game folder

### Punishment Settings
+ **Shock Intensity**: Electric shock intensity for incorrect answers (10-100)
+ **Shock Duration**: Duration of the electric shock (1-10 seconds)

### Time Settings
+ **Answer Time Limit**: Time limit for answering each question (10-300 seconds, 0 for unlimited)

### Startup Settings
+ **Manual Start**: When enabled, the game can only be started by pressing the automatic lock device button

## Question File Format
The question file should be in JSON format, containing the following structure:

```json
{
  "questions": [
    {
      "question": "Question content",
      "options": ["Option A", "Option B", "Option C", "Option D"],
      "correct": 0,
      "explanation": "Answer explanation (optional)"
    }
  ]
}
```

### Field Descriptions
+ `question`: The question text
+ `options`: Array of options
+ `correct`: Index of the correct answer (starting from 0)
+ `explanation`: Explanation of the answer (optional)

## Game Flow
1. **Game Preparation**
    - Load the question file
    - Check device connection status
    - Initialize game parameters
2. **Game Start**
    - Manual start mode: Wait for the automatic lock device button signal
    - Automatic start mode: Immediately start with the first question
3. **Answering Process**
    - Display the question and options
    - Automatic lock device locks (prevents exiting midway)
    - Start the countdown timer
    - Wait for the player to select an answer
4. **Answer Judgment**
    - **Correct Answer**: Display correct prompt, proceed to the next question
    - **Incorrect Answer or Timeout**: Trigger electric shock punishment, place the question at the end of the queue to be re-answered
5. **Game End**
    - Game ends after all questions are answered correctly
    - Automatic lock device unlocks
    - Display game statistics

## Control Methods
### Mouse Operation
+ Click on option buttons to select an answer

### QTZ Button Operation
+ **Button 0**: Cycle up through options
+ **Button 1**: Cycle down through options / Confirm selection
+ The selected option will be highlighted
