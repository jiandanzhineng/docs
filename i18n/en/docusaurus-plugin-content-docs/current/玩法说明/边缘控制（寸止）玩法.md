### Game Mechanism
1. **Pressure Monitoring**: The air pressure sensor detects anal sphincter pressure changes in real-time.
2. **Intelligent Adjustment**: Reduces stimulation when pressure is high, enhances stimulation when pressure is low.
3. **Shock Warning**: Triggers an electric shock to alert the user when pressure exceeds the critical value.
4. **Delayed Start**: Delays stimulation for a period when pressure is low before starting gentle stimulation.
5. **Dynamic Balance**: Maintains a balanced state near the critical pressure.
