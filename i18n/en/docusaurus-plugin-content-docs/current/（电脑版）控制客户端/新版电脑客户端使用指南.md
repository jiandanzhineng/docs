# New PC Client User Guide

1. Download [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2. **<font>Please extract and install after downloading. If extraction fails, install</font>** [7zip](https://www.7-zip.org/download.html) **<font>and use it to extract the file.</font>**
3. **<font>Start the client.</font>**
4. **<font>Enable the PC hotspot, set the Wi-Fi name to 'easysmart', password to '8 ones', and the band to</font>****<font>2.4G</font>****<font>band. (If there is no PC hotspot, see the section below.)</font>**
5. **<font>(Important)</font>****<font>If the device connects to the Wi-Fi but does not appear in the client, it may be blocked. You need to disable the computer firewall.</font>**
6. **<font>Power on the device. It will automatically connect to the computer about 15-30 seconds after startup.</font>**
7. **<font>You can start enjoying it now!</font>**

 

## How to Use Without PC Hotspot
1. Configure device network: Refer to [Connecting Devices to WiFi via APP](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) to connect the device to your home WiFi (Note: it must be 2.4G WiFi).
2. Enable mDNS in the client.
