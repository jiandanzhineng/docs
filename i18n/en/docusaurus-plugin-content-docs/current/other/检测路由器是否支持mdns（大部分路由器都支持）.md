# Check if the router supports mDNS (most routers support it)

Detection tool: [https://wwcg.lanzouu.com/i13Al2owjkra](https://wwcg.lanzouu.com/i13Al2owjkra)

![1740662709359-df988512-bac5-44da-9e04-dad40127cebf.png](./img/vx4ed-gCjjyFr7tF/1740662709359-df988512-bac5-44da-9e04-dad40127cebf-168097.png)

If the monitoring result shows support for mDNS, there is no problem.

If the displayed content is not like this, it cannot be used directly under the current router.

Known examples:

Huawei AX2 may not support it.

Other tested router models all support it.
