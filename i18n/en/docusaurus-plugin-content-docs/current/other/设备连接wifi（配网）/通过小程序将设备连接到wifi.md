# Connecting Devices to WiFi via Mini Program

All devices can connect to WiFi following this document.

**<font>Note: If you want to use the default WiFi, please do not configure the network. The default WiFi name is easysmart, password is 11111111. The device will automatically connect to the default WiFi.</font>**

Video Tutorials:

Douyin: [https://v.douyin.com/l_QWua94sRA/](https://v.douyin.com/l_QWua94sRA/)

YT: [https://www.youtube.com/watch?v=W7ITGIC0lw8](https://www.youtube.com/watch?v=W7ITGIC0lw8)

**<font>Prerequisites:</font>**

1.  The space has a 2.4G wireless WiFi network.
2.  The mobile phone is currently connected to that wireless WiFi (if it's a combined 2.4G/5G network with the same name, connecting to the 5G WiFi is also acceptable).

If unsuccessful, you can also [Connect Devices to WiFi via APP](./通过APP将设备连接到wifi.md).

## Step 1: Power on the device. The device should light up at this point.
Example:

![1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0.jpeg](./img/BwCUIIfLskyFx0zd/1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0-968887.jpeg)

## Step 2: Search for the WeChat mini program "物联地带蓝牙配网" and click to enter.
<font>Note: Please enable mobile phone Bluetooth before operation.</font>

After entering, it looks like the following:

![1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7.jpeg](./img/BwCUIIfLskyFx0zd/1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7-216457.jpeg)

Click "Start Network Configuration". After a few seconds, it will look like the following:

(Note: Devices can only be searched after they are powered on. If the device has successfully configured the network and connected to WiFi, it will not be searchable.)

![1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa.jpeg](./img/BwCUIIfLskyFx0zd/1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa-204739.jpeg)

Select the found device.

Enter the password for the WiFi your phone is currently connected to (must be 2.4G or a combined network with the same name for 2.4G/5G).

Click "Next".

![1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9.jpeg](./img/BwCUIIfLskyFx0zd/1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9-470214.jpeg)

The network configuration will succeed after a few seconds.

![1736778915718-1e769ec6-d814-49fe-91b6-771f19454784.jpeg](./img/BwCUIIfLskyFx0zd/1736778915718-1e769ec6-d814-49fe-91b6-771f19454784-492537.jpeg)

Even if it shows "Network Configuration Failed", it does not affect the result; it is still successful. The network configuration is complete at this point.

**<font>Note: After the device connects to WiFi, it will automatically turn off Bluetooth. If it shows failure but you cannot find the device when searching again, it means it has already succeeded.</font>**
