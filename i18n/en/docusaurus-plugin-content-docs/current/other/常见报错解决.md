# Common Error Resolution

1. Device cannot automatically go online after configuration and client startup  
   1. Confirm whether your router supports mDNS. Refer to [Checking if Router Supports mDNS (Most Routers Do)](./检测路由器是否支持mdns（大部分路由器都支持）.md). If it indicates unsupported, you can:  
      1. Connect the computer and device via mobile hotspot for playing  
      2. Check the connection relationship between your router and modem. If the router is in AP mode, change it to DHCP mode  

2. Errors related to vcruntime140.dll  

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)  

Download and install <font>Visual C++ Redistributable for Visual Studio 2015</font>  

 

2. When starting, the following appears:  

<font>Unable to load emulator DLL</font>  
<font>(C:\beidanci\emqx\erts-13.2.2.4\bin\beam.smp.dll)</font>  
<font>SET should not appear at this time.</font>  

 

<font>Solution: Install Erlang</font>
