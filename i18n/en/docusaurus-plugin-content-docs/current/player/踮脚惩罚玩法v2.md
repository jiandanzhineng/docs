# Tiptoe Penalty Gameplay v2

High-tech somatosensory self-discipline training game - Tiptoe standing challenge

### V1.4.0 Update Content:
1. **Added TD01 Device**: Prevents prolonged boredom by activating after a certain period without triggering electroshock, with intensity gradually increasing.
2. **Added Keypress Start Game Operation for auto_lock**: Convenient for single-player operation; after enabling manual start mode, the game can be started via a keypress.
3. **auto_lock and TD01 are optional devices**: The game can be played normally without these devices; only an electroshock device and a QTZ sensor are required.

## 🎮 Gameplay Details Explained
### 📋 Game Preparation
#### 1. Device Configuration
The system automatically connects four core devices:

+ 🔒 **Auto Lock**: Locks when the game starts to prevent quitting midway (optional device)
+ ⚡ **Electroshock Device**: Provides immediate feedback/punishment
+ 📡 **QTZ Laser Sensor**: Monitors whether the heel is pressed down via dual buttons (placed under the heel)
+ 🎮 **TD01 Device**: Optional enhancement device that activates after a set period without triggering electroshock, gradually increasing intensity to disrupt tiptoeing.

#### 2. Parameter Settings
Game difficulty can be customized:

+ **Game Duration**: Adjustable from 1 to 360 minutes
+ **Electroshock Intensity**: Levels 10-100
+ **Progressive Mode**: Punishment severity increases with violation count
+ **Safety Limit**: Option to allow exceeding the 30V safety limit
+ **Manual Start**: Option to wait for a keypress on the auto_lock device to start the game
+ **TD01 Delay**: Adjustable from 1 to 60 seconds; how long after no electroshock trigger before TD01 activates
+ **TD01 Intensity Increment**: Intensity increase per 5 seconds (1-100)

## 🎯 Specific Gameplay
### Core Rules
Maintain a tiptoe standing posture; heels must absolutely NOT touch the ground!

#### 1. Game Start
+ System automatically locks (if auto_lock device is present), preventing mid-game exit
+ If manual start mode is enabled, a keypress on the auto_lock device is required to start the timer
+ Timer starts; tiptoe posture must be maintained
+ QTZ sensor begins real-time monitoring of heel position

#### 2. Monitoring Mechanism
+ Sensor has two button detection points (button0 and button1)
+ Heel pressing down on either button = violation
+ System immediately triggers electroshock punishment
+ Electroshock only stops when both buttons are released

#### 3. Punishment System
+ 🔴 **Immediate Punishment**: Electroshock begins the instant the heel presses a button
+ 📈 **Progressive Intensity**: Electroshock intensity automatically increases with more violations
+ ⏱️ **Continuous Punishment**: Electroshock persists as long as any button is pressed, until fully released
+ 🎮 **TD01 Enhanced Punishment**: TD01 device activates after a specified period without electroshock, intensity increasing every 5 seconds

#### 4. TD01 Device Mechanism
+ **Delayed Activation**: Automatically activates after a specified number of seconds without electroshock trigger
+ **Intensity Increment**: Automatically increases intensity by the set value every 5 seconds
+ **Maximum Intensity**: Supports 0-255 range, unaffected by safety limits
+ **Automatic Stop**: Automatically stops and resets when electroshock is triggered

#### 5. Game End
+ Automatically unlocks upon reaching the set time
+ System displays a complete performance report
+ Statistics include violation count, electroshock count, etc.

## 💪 Challenge Points
+ **Physical Challenge**: Long-term tiptoe standing tests leg muscle endurance
+ **Willpower Test**: Maintaining standard posture despite fatigue
+ **Precision Control**: Any subtle heel lowering will be detected
+ **Psychological Game**: Maintaining calm and focus under the threat of electroshock
+ **Dual Pressure**: Must avoid electroshock while also dealing with TD01 device interference

## 🏆 Victory Conditions
Successfully persist until the set time ends; fewer violations yield better performance:

+ 🌟 **Perfect Performance**: 0 violations
+ ⭐ **Excellent Performance**: 1-2 violations
+ 👍 **Good Performance**: 3-5 violations
+ 😅 **Needs Improvement**: More than 5 violations
