Core Rule : Maintain a tiptoe standing posture, heels must absolutely not touch the ground!

1. Game Start
    - System auto-locks, no mid-game exit possible
    - Timer starts, must maintain tiptoe posture
    - QTZ sensor begins real-time heel position monitoring
2. Monitoring Mechanism
    - Sensor has two button detection points
    - Pressing either button with the heel = infraction
    - System immediately triggers e-stim punishment
    - E-stim stops only when both buttons are released
3. Punishment System
    - 🔴 Immediate Punishment : E-stim starts the moment heel presses button
    - 📈 Progressive Intensity : E-stim strength automatically increases with more infractions
    - ⏱️ Continuous Punishment : E-stim persists as long as button is pressed, stops only when fully released
    - 🎮 Dilemma Punishment: After tiptoeing for a while, activates TD01 device for interference
4. Game End
    - Automatically unlocks after reaching the set time
    - System displays a complete performance report
    - Statistics include infraction count, e-stim frequency, etc. 💪 Challenge Points
+ Physical Challenge : Long-term tiptoe standing tests leg muscle endurance
+ Willpower Test : Must maintain proper posture even when fatigued
+ Precise Control : Even the slightest heel lowering is detected
+ Psychological Game : Maintaining calm and focus under threat of e-stim 🏆 Victory Conditions  
Successfully persist until the set time ends, fewer infractions equals better performance:
+ 🌟 Perfect Performance : 0 infractions
+ ⭐ Excellent Performance : 1-2 infractions
+ 👍 Good Performance : 3-5 infractions
+ 😅 Needs Improvement : More than 5 infractions

### ⚠️ Safety Reminders
+ Ensure good health, with no heart conditions or other illnesses
+ Recommended to start with lower intensity and shorter durations
+ If experiencing discomfort, stop the game immediately
