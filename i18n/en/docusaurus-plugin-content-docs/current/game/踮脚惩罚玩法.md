Core Rule: Maintain a tiptoe standing posture, heels must absolutely not touch the ground!

1. Game Start
    - System auto-locks, cannot exit midway
    - Timer starts, must maintain tiptoe posture
    - QTZ sensor begins real-time monitoring of heel position
2. Monitoring Mechanism
    - Sensor has two button detection points
    - Pressing either button with heel = Violation
    - System immediately triggers electric shock punishment
    - Shock stops only when both buttons are released
3. Punishment System
    - 🔴 Instant Punishment: Electric shock starts the moment heel presses button
    - 📈 Progressive Intensity: Shock intensity automatically increases with more violations
    - ⏱️ Continuous Punishment: Shock persists as long as button is pressed, stops only when fully released
    - 🎮 Dilemma Punishment: After tiptoeing for a period, the TD01 device activates to cause interference
4. Game End
    - Auto-unlocks after reaching the set time
    - System displays a complete performance report
    - Statistics include violation count, shock count, etc. 💪 Challenge Points
+ Physical Challenge: Prolonged tiptoe standing tests leg muscle endurance
+ Willpower Test: Must maintain standard posture despite fatigue
+ Precise Control: Any subtle heel lowering is detected
+ Psychological Game: Staying calm and focused under the deterrence of electric shock 🏆 Victory Conditions  
Successfully persist until the set time ends; fewer violations mean better performance:
+ 🌟 Perfect Performance: 0 violations
+ ⭐ Excellent Performance: 1-2 violations
+ 👍 Good Performance: 3-5 violations
+ 😅 Needs Improvement: More than 5 violations
