# Connect Device to Wi-Fi via APP

**Note: If you want to use the default Wi-Fi, do not perform network configuration. The default Wi-Fi name is `easysmart` and password is `11111111`. The device will automatically connect to the default Wi-Fi.**

**Prerequisites:**

1.  There is a 2.4G wireless Wi-Fi network in the space.

If unsuccessful, you can also [Connect Device to Wi-Fi via Mini Program](./通过小程序将设备连接到wifi.md).

### 1. Insert batteries into the device or plug in USB power.
### 2. (Required for first-time device use) Download the network configuration app.
Android: [https://github.com/EspressifApp/EspBlufiForAndroid/releases](https://github.com/EspressifApp/EspBlufiForAndroid/releases)

Apple: [https://apps.apple.com/cn/app/espblufi/id1450614082](https://apps.apple.com/cn/app/espblufi/id1450614082)

### 3. Start network configuration using the Android client as an example.
Note: Please enable your phone's Bluetooth before operating.

1.  Open the client.

![1736779065125-88f8f145-aa31-4f23-8fc1-688135201141.jpeg](./img/seRLToJVhydnyX9B/1736779065125-88f8f145-aa31-4f23-8fc1-688135201141-782102.jpeg)

2.  Swipe down to refresh the device list.

![1736779107818-6c9229bc-b810-443e-8c97-02a92875a1cb.jpeg](./img/seRLToJVhydnyX9B/1736779107818-6c9229bc-b810-443e-8c97-02a92875a1cb-609160.jpeg)

3.  Click on the device.

![1736779081873-ae6a7216-61da-4f2f-b52c-12c4508da0fc.jpeg](./img/seRLToJVhydnyX9B/1736779081873-ae6a7216-61da-4f2f-b52c-12c4508da0fc-267799.jpeg)

4.  Click Connect.

![1736779216224-f939c610-f560-496c-a0ae-45c351df2e26.png](./img/seRLToJVhydnyX9B/1736779216224-f939c610-f560-496c-a0ae-45c351df2e26-543871.png)

5.  Click Network Configuration.

![1736779095081-463e02af-bdb3-410e-b3c1-8528afcc1dde.png](./img/seRLToJVhydnyX9B/1736779095081-463e02af-bdb3-410e-b3c1-8528afcc1dde-063427.png)

6.  Select Station mode (default), enter the Wi-Fi name and password (Note: Only 2.4G Wi-Fi is supported).

![1736779099837-569e5c08-1921-4e54-b5e5-98186de7be31.png](./img/seRLToJVhydnyX9B/1736779099837-569e5c08-1921-4e54-b5e5-98186de7be31-589046.png)

7.  If it's a dual-band unified SSID, just select Continue.

![1736779135983-6e45be09-0c84-4236-9549-688f76552344.jpeg](./img/seRLToJVhydnyX9B/1736779135983-6e45be09-0c84-4236-9549-688f76552344-071259.jpeg)

8.  After that, the setup is complete.

**Note: After the device connects to Wi-Fi, Bluetooth will automatically turn off. If it shows failure but you can no longer find the device when searching again, it means the configuration was successful.**

![1736779101106-be6bd6a9-00fe-4676-9b08-a8a5a0db7df2.jpeg](./img/seRLToJVhydnyX9B/1736779101106-be6bd6a9-00fe-4676-9b08-a8a5a0db7df2-210376.jpeg)
