# Connecting Devices to Wi-Fi via Mini Program

All devices can connect to Wi-Fi by following this document.

**<font>Note: If you want to use the default Wi-Fi, do not perform network configuration. The default Wi-Fi name is easysmart, password is 11111111. The device will automatically connect to the default Wi-Fi.</font>**

Video Tutorial:

Douyin:[https://v.douyin.com/l_QWua94sRA/](https://v.douyin.com/l_QWua94sRA/)

YT：[https://www.youtube.com/watch?v=W7ITGIC0lw8](https://www.youtube.com/watch?v=W7ITGIC0lw8)

**<font>Prerequisites:</font>**

1. There is a 2.4G wireless Wi-Fi network in the space.
2. The phone is currently connected to that wireless Wi-Fi (if it's a 2.4G/5G combined network, connecting to the 5G network with the same name is also acceptable).

If unsuccessful, you can also [Connect the Device to Wi-Fi via App](./通过APP将设备连接到wifi.md).

## Step 1: Start the device. The device should light up.
Example:

![1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0.jpeg](./img/BwCUIIfLskyFx0zd/1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0-968887.jpeg)

## Step 2: Search for the mini program "物联地带蓝牙配网" in WeChat and click to enter.
<font>Note: Please activate Bluetooth on your phone before operating.</font>

After entering, it looks like the image below:

![1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7.jpeg](./img/BwCUIIfLskyFx0zd/1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7-216457.jpeg)

Click "Start Network Configuration". After a few seconds, it will look like the image below:

(Note: The device must be powered on to be found. If the device has successfully configured the network and connected to Wi-Fi, it will not be found.)

![1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa.jpeg](./img/BwCUIIfLskyFx0zd/1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa-204739.jpeg)

Select the found device.

Enter the password of the Wi-Fi network your phone is currently connected to (must be 2.4G or share the same name with a 5G network).

Click "Next".

![1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9.jpeg](./img/BwCUIIfLskyFx0zd/1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9-470214.jpeg)

The network configuration will succeed in a few seconds.

![1736778915718-1e769ec6-d814-49fe-91b6-771f19454784.jpeg](./img/BwCUIIfLskyFx0zd/1736778915718-1e769ec6-d814-49fe-91b6-771f19454784-492537.jpeg)

If it shows configuration failure, it does not matter; it is still successful. The configuration is now complete.

**<font>Note: After the device connects to Wi-Fi, it will automatically turn off Bluetooth. If it shows failure but the device cannot be found when searching again, then it has succeeded.</font>**
