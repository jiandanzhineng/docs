# Common Error Troubleshooting

1. Device cannot automatically connect after client starts following network configuration.
    1. Please confirm if your router supports mdns. Refer to [Check if Your Router Supports mdns (Most Routers Do)](./检测路由器是否支持mdns（大部分路由器都支持）.md). If the check indicates it is not supported, you can:
        1. Connect your computer and device via a mobile hotspot to play.
        2. Check the connection relationship between your router and optical modem. If the router mode is AP, you can change it to DHCP mode.
2. vcruntime140.dll related error.

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

Download and install <font>Visual C++ Redistributable for Visual Studio 2015</font>.
