# Configuración de red a través del cliente del PC (se requiere que el PC tenga Bluetooth)

<font>Nota: Este método aún está en prueba y puede ser inestable. Si falla, intente finalizar el proceso de configuración y reiniciarlo, o utilice otros métodos.</font>

**<font>Nota: Si desea usar la red Wi-Fi predeterminada, no realice la configuración. El nombre de la red Wi-Fi predeterminada es easysmart, la contraseña es 11111111, y el dispositivo se conectará automáticamente a la red Wi-Fi predeterminada.</font>**

![1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4.png](./img/hURz820DTNcJYUCH/1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4-096925.png)

1. Abra el cliente, haga clic en Gestión de dispositivos y luego en Configuración de red del dispositivo.

![1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed.png](./img/hURz820DTNcJYUCH/1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed-155975.png)

2. Ingrese el nombre de la red Wi-Fi y la contraseña (tenga cuidado de no ingresar errores, de lo contrario, el dispositivo no podrá conectarse a Internet incluso si la configuración de red se realiza con éxito).
3. Haga clic en Modificar configuración.
4. Haga clic en Iniciar configuración de red para comenzar el proceso.
