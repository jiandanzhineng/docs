# Conectar dispositivos a WiFi mediante miniprograma

Todos los dispositivos pueden conectarse a WiFi siguiendo este documento.

**<font>Nota: Si desea utilizar la WiFi predeterminada, no realice la configuración de red. El nombre de la WiFi predeterminada es easysmart y la contraseña es 11111111. El dispositivo se conectará automáticamente a la WiFi predeterminada.</font>**

Video tutorial:

Douyin:[https://v.douyin.com/l_QWua94sRA/](https://v.douyin.com/l_QWua94sRA/) 

YT：[https://www.youtube.com/watch?v=W7ITGIC0lw8](https://www.youtube.com/watch?v=W7ITGIC0lw8)

**<font>Requisitos previos:</font>**

1. El espacio debe tener una red WiFi inalámbrica de 2.4G.
2. El teléfono móvil debe estar conectado actualmente a esta red WiFi (si es 2.4G y 5G combinados, también es aceptable conectarse a la red 5G con el mismo nombre).

Si no puede tener éxito, también puede [conectar el dispositivo a WiFi mediante la APP](./通过APP将设备连接到wifi.md)

## Paso 1: Encienda el dispositivo. El dispositivo debe encender la luz en este momento.
Ejemplo:

![1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0.jpeg](./img/BwCUIIfLskyFx0zd/1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0-968887.jpeg)

## Paso 2: Busque el miniprograma "物联地带蓝牙配网" en WeChat y haga clic para ingresar.
<font>Nota: Active el Bluetooth del teléfono antes de operar.</font>

Después de ingresar, se verá como en la siguiente imagen:

![1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7.jpeg](./img/BwCUIIfLskyFx0zd/1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7-216457.jpeg)

Haga clic en "Comenzar configuración de red". Después de unos segundos, se verá como en la siguiente imagen:

(Nota: Solo podrá buscar el dispositivo después de encenderlo.

Si el dispositivo se configura exitosamente y se conecta a WiFi, ya no podrá buscarlo.)

![1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa.jpeg](./img/BwCUIIfLskyFx0zd/1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa-204739.jpeg)

Seleccione el dispositivo encontrado.

Ingrese la contraseña de la WiFi a la que está conectado actualmente el teléfono (debe ser 2.4G o 2.4 y 5 con el mismo nombre).

Haga clic en "Siguiente".

![1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9.jpeg](./img/BwCUIIfLskyFx0zd/1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9-470214.jpeg)

Después de unos segundos, la configuración de red se completará exitosamente.

![1736778915718-1e769ec6-d814-49fe-91b6-771f19454784.jpeg](./img/BwCUIIfLskyFx0zd/1736778915718-1e769ec6-d814-49fe-91b6-771f19454784-492537.jpeg)

Si muestra "Configuración de red fallida", no afecta. También es exitoso. En este punto, la configuración de red está completa.

**<font>Nota: Una vez que el dispositivo se conecta a WiFi, el Bluetooth se apagará automáticamente. Si muestra falla pero al buscar nuevamente no encuentra el dispositivo, significa que ya fue exitoso.</font>**
