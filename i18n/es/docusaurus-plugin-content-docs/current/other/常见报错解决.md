# Solución de errores comunes

1. El cliente no se conecta automáticamente después de la configuración del dispositivo y el inicio del cliente
    1. Verifique si su enrutador soporta mdns, consulte [Detectar si el enrutador soporta mdns (la mayoría de los enrutadores lo soportan)](./检测路由器是否支持mdns（大部分路由器都支持）.md). Si indica que no es compatible, puede:
        1. Jugar conectando la computadora y el dispositivo a través del punto de acceso del teléfono móvil.
        2. Verificar la conexión entre su enrutador y módem. Si el modo del enrutador es AP, puede cambiarlo a modo DHCP.
2. Error relacionado con vcruntime140.dll

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

Descargar e instalar <font>Visual C++ Redistributable for Visual Studio 2015</font>
