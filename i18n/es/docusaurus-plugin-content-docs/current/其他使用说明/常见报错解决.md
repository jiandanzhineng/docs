# Resolución de Errores Comunes

1. El cliente no se conecta automáticamente después de iniciarse, una vez configurado el dispositivo
    1. Verifica si tu router es compatible con mdns. Consulta [Cómo verificar si el router soporta mdns (la mayoría de los routers lo hacen)](./检测路由器是否支持mdns（大部分路由器都支持）.md). Si la verificación indica que no es compatible, puedes:
        1. Conectar tu computadora y el dispositivo a través de un punto de acceso (hotspot) del teléfono para jugar.
        2. Verificar la relación de conexión entre tu router y el módem. Si el modo del router está configurado como AP, cámbialo al modo DHCP.
2. Errores relacionados con vcruntime140.dll

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id?id=48145)

Descarga e instala **Visual C++ Redistributable for Visual Studio 2015**

2. Al iniciar, aparece el error:
