# Conectar el dispositivo a WiFi mediante APP

**<font>Nota: Si desea usar la WiFi por defecto, no configure la red. El nombre de la WiFi por defecto es easysmart, la contraseña es 11111111. El dispositivo se conectará automáticamente a la WiFi por defecto.</font>**

**<font>Prerrequisito:</font>**

1.  Que exista una red WiFi inalámbrica de 2.4G en el espacio.

Si no se puede conectar exitosamente, también puede [conectar el dispositivo a WiFi a través de una mini aplicación (applet)](./通过小程序将设备连接到wifi.md)

### <font>1. Colocar las baterías en el dispositivo o conectar la alimentación USB</font>
### <font>2. (Necesario para el primer uso del dispositivo) Descargar la aplicación de configuración de red</font>
<font>Android:</font> [<font>https://github.com/EspressifApp/EspBlufiForAndroid/releases</font>](https://github.com/EspressifApp/EspBlufiForAndroid/releases)

<font>Apple:</font> [<font>https://apps.apple.com/cn/app/espblufi/id1450614082</font>](https://apps.apple.com/cn/app/espblufi/id1450614082)

### <font>3. Tomando el cliente Android como ejemplo, comenzar la configuración de red</font>
<font>Nota: Active el Bluetooth del teléfono antes de operar.</font>

1.  <font>Abrir la aplicación cliente</font>

![1736779065125-88f8f145-aa31-4f23-8fc1-688135201141.jpeg](./img/seRLToJVhydnyX9B/1736779065125-88f8f145-aa31-4f23-8fc1-688135201141-782102.jpeg)

1.  <font>Deslizar hacia abajo para actualizar la lista de dispositivos</font>

![1736779107818-6c9229bc-b810-443e-8c97-02a92875a1cb.jpeg](./img/seRLToJVhydnyX9B/1736779107818-6c9229bc-b810-443e-8c97-02a92875a1cb-609160.jpeg)

1.  <font>Hacer clic en el dispositivo</font>

![1736779081873-ae6a7216-61da-4f2f-b52c-12c4508da0fc.jpeg](./img/seRLToJVhydnyX9B/1736779081873-ae6a7216-61da-4f2f-b52c-12c4508da0fc-267799.jpeg)

1.  <font>Hacer clic en "Conectar"</font>

![1736779216224-f939c610-f560-496c-a0ae-45c351df2e26.png](./img/seRLToJVhydnyX9B/1736779216224-f939c610-f560-496c-a0ae-45c351df2e26-543871.png)

1.  <font>Hacer clic en "Configurar red"</font>

![1736779095081-463e02af-bdb3-410e-b3c1-8528afcc1dde.png](./img/seRLToJVhydnyX9B/1736779095081-463e02af-bdb3-410e-b3c1-8528afcc1dde-063427.png)

1.  <font>Seleccionar el modo Station (es el modo por defecto), introducir el nombre (SSID) y la contraseña de la WiFi (Nota: Solo se admite WiFi de 2.4G)</font>

![1736779099837-569e5c08-1921-4e54-b5e5-98186de7be31.png](./img/seRLToJVhydnyX9B/1736779099837-569e5c08-1921-4e54-b5e5-98186de7be31-589046.png)

1.  <font>Si la red tiene bandas duales combinadas (dual-band合一), seleccionar "Continuar"</font>

![1736779135983-6e45be09-0c84-4236-9549-688f76552344.jpeg](./img/seRLToJVhydnyX9B/1736779135983-6e45be09-0c84-4236-9549-688f76552344-071259.jpeg)

1.  <font>Después de eso, el proceso está completo</font>

**<font>Nota: Una vez que el dispositivo se conecte a la WiFi, el Bluetooth se apagará automáticamente. Si muestra un error pero ya no puede encontrar el dispositivo al buscar de nuevo, significa que la configuración fue exitosa.</font>**
