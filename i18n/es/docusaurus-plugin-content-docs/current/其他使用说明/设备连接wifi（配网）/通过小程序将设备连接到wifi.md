# Conectar dispositivos a WiFi mediante mini programa

Todos los dispositivos pueden conectarse a WiFi siguiendo este documento.

**<font>Nota: Si desea usar el WiFi predeterminado, no realice la configuración de red. El nombre del WiFi predeterminado es easysmart, la contraseña es 11111111. El dispositivo se conectará automáticamente al WiFi predeterminado.</font>**

Tutorial en video:

Douyin: [https://v.douyin.com/l_QWua94sRA/](https://v.douyin.com/l_QWua94sRA/)

YT: [https://www.youtube.com/watch?v=W7ITGIC0lw8](https://www.youtube.com/watch?v=W7ITGIC0lw8)

**<font>Requisitos previos:</font>**

1.  El espacio cuenta con WiFi inalámbrico de 2.4G.
2.  El teléfono móvil está actualmente conectado a esa red WiFi inalámbrica (si es 2.4G y 5G combinados, también se puede conectar a la red WiFi 5G con el mismo nombre).

Si no puede tener éxito, también puede [Conectar dispositivos a WiFi mediante APP](./通过APP将设备连接到wifi.md).

## Paso 1: Encender el dispositivo. El dispositivo debería encender su luz en este momento.
Ejemplo:

![1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0.jpeg](./img/BwCUIIfLskyFx0zd/1736778961469-1ea5e8a0-5131-4123-809d-c12aa7e32ee0-968887.jpeg)

## Paso 2: Buscar en WeChat el mini programa "物联地带蓝牙配网" y hacer clic para entrar.
<font>Nota: Active el Bluetooth del teléfono antes de operar.</font>



Después de entrar, se verá como en la siguiente imagen.

![1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7.jpeg](./img/BwCUIIfLskyFx0zd/1736778759468-c1415188-e182-4a02-8bcb-661d1d3510a7-216457.jpeg)

Haga clic en "Comenzar configuración de red". Después de unos segundos, se verá como en la siguiente imagen.

(Nota: Solo se puede buscar después de encender el dispositivo.

Si la configuración de red del dispositivo es exitosa y se conecta con éxito al WiFi, tampoco se podrá buscar).

![1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa.jpeg](./img/BwCUIIfLskyFx0zd/1736778774091-dfcd4838-0150-4f73-b6aa-980a7cb0e1fa-204739.jpeg)

Seleccione el dispositivo encontrado.

Ingrese la contraseña del WiFi al que está conectado actualmente su teléfono móvil (debe ser 2.4G o 2.4 y 5 con el mismo nombre).

Haga clic en "Siguiente".

![1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9.jpeg](./img/BwCUIIfLskyFx0zd/1736778837887-f0b18aa3-c9e7-4aa8-8470-08a1b62032c9-470214.jpeg)

Después de unos segundos, la configuración de red será exitosa.

![1736778915718-1e769ec6-d814-49fe-91b6-771f19454784.jpeg](./img/BwCUIIfLskyFx0zd/1736778915718-1e769ec6-d814-49fe-91b6-771f19454784-492537.jpeg)

Si muestra que la configuración de red falló, no afecta. También es exitosa. En este momento, la configuración de red está completa.

**<font>Nota: Después de que el dispositivo se conecte al WiFi, el Bluetooth se apagará automáticamente. Si muestra fallo pero al buscar de nuevo no encuentra el dispositivo, significa que ya fue exitoso.</font>**
