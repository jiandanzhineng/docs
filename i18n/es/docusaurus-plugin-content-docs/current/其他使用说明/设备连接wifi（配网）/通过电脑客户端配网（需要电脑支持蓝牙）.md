# Configuración de red a través del cliente de computadora (requiere que la computadora admita Bluetooth)

<font>Nota: Este método aún está en pruebas y puede ser inestable. Si falla, intente finalizar y reiniciar la configuración de red o utilice otro método.</font>

**<font>Nota: Si desea utilizar la red WiFi predeterminada, no configure la red. El nombre de la red WiFi predeterminada es easysmart y la contraseña es 11111111. El dispositivo se conectará automáticamente a la red WiFi predeterminada.</font>**

![1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4.png](./img/hURz820DTNcJYUCH/1758201848446-fb959d3a-a790-42c6-befe-ccbb3a9b45f4-096925.png)

1. Abra el cliente y haga clic en **Gestión de dispositivos** -> **Configuración de red del dispositivo**.

![1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed.png](./img/hURz820DTNcJYUCH/1758201881242-3be10794-3bed-47a5-998f-25a00a8e22ed-155975.png)

2. Ingrese el nombre de la red WiFi y la contraseña (asegúrese de no cometer errores, de lo contrario, la conexión a Internet fallará incluso después de una configuración exitosa).
3. Haga clic en **Modificar configuración**.
4. Haga clic en **Iniciar configuración de red** para comenzar el proceso.
