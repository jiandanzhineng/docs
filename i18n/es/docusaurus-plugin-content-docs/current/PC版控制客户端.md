# Guía de Uso del Cliente para PC

1.  Enlace de descarga: [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2.  **Por favor, descomprima el archivo antes de instalarlo. Si falla la descompresión, puede instalar [7zip](https://www.7-zip.org/download.html) y usar ese software para descomprimir.**
3.  **Inicie el cliente.**
4.  **Active el punto de acceso (hotspot) de la computadora, configure el nombre de la red WiFi como `easysmart`, la contraseña como `11111111` (ocho unos) y la banda como **2.4G** (consulte [Método de uso sin punto de acceso de computadora](#método-de-uso-sin-punto-de-acceso-de-computadora)).**
5.  **(Importante) Si el dispositivo se conecta al WiFi pero no aparece en el cliente, es posible que esté siendo bloqueado. Es necesario desactivar el firewall de la computadora.**
6.  **Encienda el dispositivo. Aproximadamente 15-30 segundos después de encenderse, se conectará automáticamente a la computadora.**
7.  **¡Ya puede comenzar a disfrutar!**

## Método de uso sin punto de acceso de computadora

1.  Configurar la red del dispositivo: Consulte [Conectar el dispositivo a WiFi mediante la APP](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) para conectar el dispositivo a la red WiFi de su hogar (nota: debe ser una red WiFi de 2.4G).
2.  Habilite mDNS en el cliente.
