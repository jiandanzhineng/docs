# Guía de Uso del Nuevo Cliente para PC

1.  Descarga [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2.  **<font>Por favor, descomprime el archivo antes de instalar. Si falla la descompresión, puedes instalar</font>**[7zip](https://www.7-zip.org/download.html)**<font> y luego usar ese software para descomprimir.</font>**
3.  **<font>Inicia el cliente.</font>**
4.  **<font>Habilita el punto de acceso (hotspot) de la computadora, configura el nombre de la red WiFi como `easysmart`, la contraseña como `11111111` (ocho unos) y la banda como</font>****<font>2.4G</font>****<font> (si no tienes la función de hotspot en tu PC, consulta más adelante).</font>**
5.  **<font>(Importante)</font>****<font>Si el dispositivo se conecta al WiFi pero no aparece en el cliente, es posible que esté siendo bloqueado. Necesitas desactivar el firewall de la computadora.</font>**
6.  **<font>Enciende el dispositivo. Aproximadamente 15-30 segundos después de iniciarse, se conectará automáticamente a la computadora.</font>**
7.  **<font>¡Ya puedes empezar a disfrutar!</font>**

## Método de Uso sin Punto de Acceso (Hotspot) en la PC
1.  Configura la red del dispositivo: Consulta [Conectar el dispositivo a WiFi mediante la APP](./其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) para conectar el dispositivo al WiFi de tu casa (debe ser una red WiFi de 2.4G).
2.  Habilita mDNS en el cliente.
