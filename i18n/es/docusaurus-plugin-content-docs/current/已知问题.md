# Problemas Conocidos

1. En algunos routers, la configuración de red del dispositivo puede fallar. En ese caso, se puede utilizar el teléfono móvil para crear una red Wi-Fi.
2. Si el dispositivo recién recibido falla en la configuración de red o no se conecta después de configurarlo, puede deberse a falta de batería. Se recomienda cargarlo o mantenerlo conectado a la corriente, y luego reiniciarlo.
3. Asegúrese de iniciar mdns solo en condiciones de red estables; de lo contrario, podría causar anomalías en la vinculación de nombres de dominio mdns (por ejemplo, en entornos de hotspot móvil, primero conéctese al hotspot y luego inicie mdns).
4. Si el botón de configuración de red aparece en gris después de conectar el teléfono al dispositivo, intente reiniciar el teléfono.
5. En modo de conexión por hotspot, debido a la calidad de la tarjeta de red, pueden ocurrir retrasos breves en las acciones periódicamente.
