# Modo de juego de entrenamiento de elevación pélvica

# Introducción al juego
+ El entrenamiento de elevación pélvica se realiza alternando las fases de "relajación" y "elevación pélvica"
+ El objetivo es completar la mayor cantidad de "elevaciones pélvicas exitosas" posible dentro del tiempo establecido
+ El éxito o el fracaso se indican visualmente en la interfaz; el fracaso activará una descarga eléctrica (si está conectado y habilitado)

# Dispositivos y preparación
+ Dispositivo obligatorio: `Sensor de presión de aire (QIYA)`
+ Dispositivos opcionales: `Dispositivo de descarga eléctrica (DIANJI)`, `Cerradura automática (ZIDONGSUO)`
+ La cerradura se activa automáticamente al inicio y se desbloquea al finalizar (si está conectada la cerradura automática)

# Explicación de parámetros
+ `Duración (minutos)`: Duración total de la sesión de juego.
+ `Objetivo de repeticiones`: Número de elevaciones exitosas que se espera completar, utilizado para mostrar el progreso general.
+ `Cambio de presión (kPa)`: Umbral de incremento de presión que debe alcanzarse en la fase de elevación pélvica (en relación a la presión mínima de la fase de relajación).
+ `Intensidad de descarga (V)`: Intensidad de la descarga eléctrica en caso de fallo.
+ `Duración de la descarga (segundos)`: Tiempo que dura la descarga eléctrica en caso de fallo.
+ `Tiempo por ciclo (segundos)`: Duración de cada fase. Por defecto es 10 segundos (relajación 10s → elevación pélvica 10s → ciclo).

# Flujo del juego
+ Fase de relajación
    - Relájate y respira naturalmente. El sistema registrará la "presión mínima" de esta fase como referencia.
    - Al terminar la cuenta regresiva, se pasa a la fase de elevación pélvica.
+ Fase de elevación pélvica
    - Debes aumentar la presión hasta alcanzar "presión mínima + cambio de presión" dentro del tiempo de esta fase.
    - Alcanzar el objetivo se considera "logrado", pero aún debes esperar a que termine la cuenta regresiva de la fase para pasar a la siguiente ronda.
    - Si no se alcanza el objetivo antes de que termine la fase, se considera "Desafío fallido · Iniciando descarga" (si el dispositivo de descarga está disponible).
+ Alternancia de fases
    - Cada ronda sigue el ciclo "relajación → elevación pélvica → relajación → ..." hasta que se acabe el tiempo o se alcance el número objetivo de repeticiones.

# Indicaciones en la interfaz
+ En la parte superior, texto grande muestra la fase actual (Fase de relajación / Fase de elevación pélvica) y el tiempo restante en esta fase.
+ Progreso general
    - Barra de progreso por repeticiones: Número de repeticiones completadas / Objetivo de repeticiones.
    - Barra de progreso por tiempo: Tiempo transcurrido / Duración total.
+ Presión y objetivo
    - Presión actual (kPa)
    - Presión mínima durante la relajación (valor de referencia)
    - Presión objetivo para la elevación pélvica (presión mínima + cambio de presión)
+ Indicación de éxito/fracaso
    - Éxito: Se muestra una cápsula verde "Logrado" después del número.
    - Fracaso: Se muestra una cápsula roja "Desafío fallido · Iniciando descarga" después del número.
+ Operaciones y registro
    - Botones: Pausar, Descarga manual.
    - Registro: Muestra información reciente y mensajes del sistema.

**Finalización y estadísticas**

+ La sesión termina al alcanzar la duración establecida o cuando el número de éxitos alcanza el objetivo.
+ Se desbloquea automáticamente al finalizar (si está conectada la cerradura automática).
+ La interfaz muestra el número acumulado de éxitos, descargas eléctricas, etc.

**Recomendaciones de uso**

+ Para usuarios primerizos, se recomienda establecer un valor bajo para "Cambio de presión" para familiarizarse con el ritmo.
+ Si se conecta un dispositivo de descarga eléctrica, comenzar con baja intensidad y ajustar gradualmente.
+ Mantener una respiración estable y concentrarse en aumentar la presión hasta el objetivo durante la fase de elevación pélvica.

**Guía rápida**

+ Conectar el sensor de presión de aire (opcionalmente conectar dispositivo de descarga, cerradura automática).
+ Configurar los parámetros y comenzar.
+ En la fase de relajación, no hacer fuerza, esperar la cuenta regresiva. En la fase de elevación pélvica, hacer fuerza para alcanzar el objetivo.
+ Al ver "Logrado", mantener hasta el final de la fase. En caso de fallo, se indicará y se aplicará descarga.
+ Repetir el ciclo hasta el final. Consultar las estadísticas y barras de progreso para conocer los resultados.
