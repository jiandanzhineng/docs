Regla Principal: ¡Mantén la postura de puntillas, los talones absolutamente no deben tocar el suelo!

1.  Inicio del Juego
    - El sistema se bloquea automáticamente, no se puede salir a mitad.
    - Comienza el conteo de tiempo, debes mantener la postura de puntillas.
    - El sensor QTZ comienza a monitorear en tiempo real la posición del talón.
2.  Mecanismo de Monitoreo
    - El sensor tiene dos puntos de detección (botones).
    - Presionar cualquiera de los botones con el talón = Infracción.
    - El sistema activa inmediatamente el castigo de electrochoque.
    - El electrochoque solo se detiene cuando AMBOS botones no están presionados.
3.  Sistema de Castigo
    - 🔴 Castigo Inmediato: El electrochoque comienza en el instante en que el talón presiona un botón.
    - 📈 Intensidad Progresiva: Cuantas más infracciones, mayor la intensidad del electrochoque automáticamente.
    - ⏱️ Castigo Continuo: El electrochoque continúa mientras el botón esté presionado, hasta que se levante completamente.
    - 🎮 Castigo de Dilema: Después de estar de puntillas un tiempo, se activa el dispositivo TD01 para crear interferencias.
4.  Fin del Juego
    - Se desbloquea automáticamente al cumplirse el tiempo establecido.
    - El sistema muestra un informe completo del desempeño.
    - Estadísticas como número de infracciones, veces electrocutado, etc. 💪 Puntos Clave del Desafío
+  Desafío Físico: Pararse de puntillas durante mucho tiempo pone a prueba la resistencia de los músculos de las piernas.
+  Prueba de Voluntad: Mantener la postura correcta a pesar de la fatiga.
+  Control Preciso: Cualquier descenso mínimo del talón será detectado.
+  Juego Mental: Mantener la calma y la concentración bajo la amenaza del electrochoque. 🏆 Condición de Victoria  
Soporta con éxito hasta el final del tiempo establecido, cuantas menos infracciones, mejor el desempeño:
+ 🌟 Desempeño Perfecto: 0 infracciones
+ ⭐ Desempeño Excelente: 1-2 infracciones
+ 👍 Desempeño Bueno: 3-5 infracciones
+ 😅 Necesita Mejorar: Más de 5 infracciones

### ⚠️ Recordatorio de Seguridad
+ Asegúrate de estar saludable, sin enfermedades cardíacas u otras afecciones.
+ Se recomienda comenzar con intensidad baja y tiempos cortos.
+ Si sientes malestar durante el juego, detente inmediatamente.
+ Solo para adultos mayores de 18 años.
