# Instrucciones de uso de Vocabulario con Estímulo por Punzada

# Descripción del método de uso:
Siempre da sueño cuando se estudian palabras

En este momento, se necesita un poco de estímulo para eliminar la somnolencia

Basta con colocar este dispositivo en el cuerpo para darle un poco de energía cuando se equivoque al estudiar~

Así la eficiencia aumentará drásticamente

# Requisitos previos de uso
1. Tener WiFi de 2.4G en casa
2. El WiFi y la computadora están en la misma red local (mismo router)
3. El router es compatible con mdns (si no es compatible, se necesita usar el punto de acceso del teléfono móvil) [Verificar si el router es compatible con mdns (la mayoría de los routers son compatibles)](../其他使用说明/检测路由器是否支持mdns（大部分路由器都支持）.md)
4. Ya se tiene el terminal de pulso inteligente simple: [Terminal de pulso inteligente simple (electrochoque)](../设备介绍&购买/简单智能脉冲（电击）终端.md) o [Enlace de Taobao](https://item.taobao.com/item.htm?id=892309919507)

## Configurar red del dispositivo
(Se recomienda cargar el dispositivo antes del primer uso)

1. Activar el dispositivo deslizando el interruptor. Una vez encendido, se encenderá la luz del dispositivo.
2. Iniciar la mini aplicación para configurar la red del dispositivo

Para este paso, ver [Conectar el dispositivo a WiFi mediante APP](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) o [Conectar el dispositivo a WiFi mediante mini aplicación](../其他使用说明/设备连接wifi（配网）/通过小程序将设备连接到wifi.md)

## Ejecutar el servidor en la computadora
1. Ejecutar el archivo "paso2启动.bat" en la computadora. La ejecución se completará en aproximadamente 2 minutos (la primera ejecución lleva más tiempo). La aparición del contenido del recuadro rojo indica que la ejecución fue exitosa (si no tiene este archivo, descárguelo en la cabecera del documento)

![1736779389985-11c305ce-2fc8-4bf3-8752-0243bff9d6c5.png](./img/thpYQ8NHwnt7wx2c/1736779389985-11c305ce-2fc8-4bf3-8752-0243bff9d6c5-503471.png)

1. Se pueden conectar los electrodos al cuerpo e insertarlos en el dispositivo.
