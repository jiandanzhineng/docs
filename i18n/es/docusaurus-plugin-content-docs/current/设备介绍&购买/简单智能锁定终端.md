Descripción de funciones:

Conecta a wifi, soporta control inalámbrico para bloquear/desbloquear el terminal, carga por USB tipo-C.

Mantener pulsado el botón durante 60 segundos para desbloqueo forzado.

Se desbloquea automáticamente cuando el voltaje es inferior al 20%.

Dirección del interruptor: Alejarse del puerto de carga es ENCENDIDO, de lo contrario es APAGADO.

Atención:

1. ¡Por favor, tenga herramientas de escape a mano durante el uso! Debido a problemas de red, etc., ¡el dispositivo no puede garantizar el desbloqueo!
