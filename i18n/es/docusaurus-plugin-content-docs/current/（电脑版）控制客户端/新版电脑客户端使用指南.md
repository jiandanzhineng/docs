# Guía de uso de la nueva versión del cliente de escritorio

1.  Descarga [https://update.ezsapi.top/control-panel/UnderSilicon.zip](https://update.ezsapi.top/control-panel/UnderSilicon.zip)
2.  **<font>Por favor, descomprime antes de instalar. Si la descompresión falla, puedes instalar</font>**[7zip](https://www.7-zip.org/download.html)**<font> y luego usar este software para descomprimir.</font>**
3.  **<font>Inicia el cliente.</font>**
4.  **<font>Habilita el punto de acceso del ordenador, configura el nombre de la red WiFi como 'easysmart', la contraseña como ocho '1' y la banda como</font>****<font>2.4G</font>****<font> (si no tienes punto de acceso en el ordenador, consulta más adelante).</font>**
5.  **<font>(Importante)</font>****<font>Si el dispositivo se conecta al WiFi pero el cliente no lo detecta, significa que está siendo bloqueado y necesitas desactivar el firewall del ordenador.</font>**
6.  **<font>Enciende el dispositivo. Aproximadamente 15-30 segundos después de encenderse, se conectará automáticamente al ordenador.</font>**
7.  **<font>¡Ya puedes empezar a disfrutar!</font>**



## Modo de uso cuando no hay punto de acceso en el ordenador

1.  Configuración de red del dispositivo: Consulta [Conectar el dispositivo a WiFi mediante la APP](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) para conectar el dispositivo a la red WiFi de tu casa (debe ser una red WiFi de 2.4G).
2.  Habilita mDNS en el cliente.
