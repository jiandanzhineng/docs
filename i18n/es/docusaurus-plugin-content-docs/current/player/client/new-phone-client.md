# Descarga del cliente

Actualmente solo es compatible con dispositivos Android, iOS estará disponible próximamente.

Enlace de descarga para Android: https://dl.undersilicon.com/apk/latest/app-release.apk

# Conexión del dispositivo (tomando cunzhi01 como ejemplo)

1. Enciende el dispositivo. En este momento, el dispositivo debería tener la luz azul parpadeando (si no parpadea, cárgalo primero).
2. Abre la aplicación cliente y haz clic en "Agregar dispositivo" o en el signo más en la esquina superior derecha.
   ![Captura de pantalla de la página principal de la aplicación ##h40vh##](./img/client/app.jpg)
3. Haz clic en "Agregar dispositivo".
   ![Descubrir dispositivo ##h40vh##](./img/client/finddevice.jpg)
4. Haz clic en "Completar".
   ![Agregar dispositivo ##h40vh##](./img/client/adddevice.jpg)
5. La adición se completa correctamente y la conexión se ha establecido correctamente. Después de una conexión exitosa, la luz azul permanecerá encendida durante unos segundos antes de apagarse.
   ![Completado ##h40vh##](./img/client/finish.jpg)
