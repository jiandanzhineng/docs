# Guía de uso del cliente para PC

1. Enlace de descarga: [https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip](https://firmware.undersilicon.cn/control-panel/stable/UnderSilicon.zip)
2. **Por favor, extrae el archivo y luego instala (si falla la extracción, puedes instalar [7zip](https://www.7-zip.org/download.html) y usar ese software para extraer)**)
3. **Inicia el cliente**
4. **Habilita el punto de acceso (hotspot) de la computadora, configura el nombre del wifi como `easysmart`, la contraseña como 8 unos (`11111111`) y la banda como **banda de 2.4G** (consulta [Método de uso sin punto de acceso de computadora](#无电脑热点情况使用方式))**  
Nota: Si el dispositivo ya está conectado a otro wifi de 5G, puede hacer que el dispositivo permanezca en la banda de 5G incluso después de configurar 2.4G. En ese caso, el dispositivo necesita conectarse a un wifi de 2.4G (o por cable de red) para configurar correctamente el punto de acceso wifi.
5. **Si el dispositivo se conecta al wifi pero no aparece en el cliente, es posible que esté bloqueado; necesitas desactivar el firewall de la computadora.**
6. **Enciende el dispositivo. Aproximadamente 15-30 segundos después de iniciarse, se conectará automáticamente a la computadora.**
7. **¡Ya puedes comenzar a disfrutar!**

 

## Método de uso sin punto de acceso de computadora {#无电脑热点情况使用方式}
1. Configurar la red del dispositivo: consulta [Conectar el dispositivo a WiFi mediante la APP](../../other/设备连接wifi（配网）/通过APP将设备连接到wifi.md) para conectar el dispositivo al wifi de tu hogar (debe ser wifi de 2.4G).
2. Habilita mDNS en el cliente.
