# Instrucciones de juego del Método "Afinar la Espalda" para Aprender Vocabulario

# Instrucciones de juego:
Siempre tienes sueño al aprender vocabulario

En ese momento, necesitas un pequeño estímulo para eliminar la somnolencia

Solo coloca este dispositivo en tu cuerpo, y te dará un impulso de energía cuando cometas un error~

¡Así tu eficiencia aumentará drásticamente!

# Requisitos previos
1. Tener wifi de 2.4GHz en casa.
2. El wifi y la computadora están en la misma red local (conectados al mismo router).
3. El router debe soportar mdns (si no lo soporta, necesitas usar el hotspot del móvil) [Verificar si el router soporta mdns (la mayoría de los routers lo soportan)](../other/检测路由器是否支持mdns（大部分路由器都支持）.md).
4. Ya tener un terminal simple de pulso inteligente: [Terminal simple de pulso (descarga eléctrica) inteligente](../device/简单智能脉冲（电击）终端.md) o [Enlace de Taobao](https://item.taobao.com/item.htm?id=892309919507).

## Configurar el dispositivo en la red
(Se recomienda cargar el dispositivo antes del primer uso)

1. Activa el dispositivo moviendo el interruptor. Después de encenderse, el dispositivo se iluminará.
2. Inicia la mini aplicación para configurar el dispositivo en la red.

Para este paso, consulta [Conectar el dispositivo a WiFi a través de la APP](../other/设备连接wifi（配网）/通过APP将设备连接到wifi.md) o [Conectar el dispositivo a WiFi a través de la mini aplicación](../other/设备连接wifi（配网）/通过小程序将设备连接到wifi.md).

## Ejecutar el servidor en la computadora
1. En la computadora, ejecuta "步骤2启动.bat". Se iniciará aproximadamente en 2 minutos (la primera ejecución lleva más tiempo). Cuando aparezca el contenido del marco rojo, significa que se ha iniciado correctamente (si no tienes este archivo, descárgalo desde el encabezado del documento).

![1736779389985-11c305ce-2fc8-4bf3-8752-0243bff9d6c5.png](./img/thpYQ8NHwnt7wx2c/1736779389985-11c305ce-2fc8-4bf3-8752-0243bff9d6c5-503471.png)

1. Puedes conectar los electrodos a tu cuerpo e insertarlos en el dispositivo.
