# Juego de Castigo por Puntas de Pie

## 🎭 Juego de Castigo por Perezosa de Criada v1.4.0
Juego de entrenamiento de autodisciplina con alta tecnología de detección de movimiento - Desafío de pararse de puntillas

### 🎮 Detalles del Juego 📋 Preparación del Juego
1. Configuración del equipo**: El sistema conecta automáticamente tres dispositivos principales
    - 🔒 Cerradura automática: Se bloquea al inicio del juego, evitando salidas prematuras [Terminal de Bloqueo Inteligente Simple](../device/简单智能锁定终端.md)
    - ⚡ Dispositivo de electrochoque: Proporciona retroalimentación de castigo inmediata [Terminal de Pulso Inteligente Simple (Electrochoque)](../device/简单智能脉冲（电击）终端.md)
    - 📡 Sensor láser QTZ: Monitorea si el talón está presionado a través de una plataforma (colocado debajo del talón) [Terminal de Detección de Distancia Inteligente Simple](../device/简单智能距离感应终端.md)

+ Configuración de parámetros**: Personaliza la dificultad del juego
    - Duración del juego: Ajustable de 1 a 180 minutos
    - Intensidad del electrochoque: Niveles de 10 a 100
    - Modo progresivo: A mayor número de infracciones, mayor severidad del castigo 🎯 Cómo Jugar

Regla central**: ¡Mantén la postura de puntillas, los talones absolutamente NO deben tocar el suelo!

1. Inicio del juego
    - El sistema se bloquea automáticamente, imposibilitando salir a mitad del juego
    - Comienza la cuenta regresiva, debes mantener la postura de puntillas
    - El sensor QTZ comienza a monitorear en tiempo real la posición del talón
2. Mecanismo de monitoreo
    - El sensor tiene dos puntos de detección/botones
    - El talón presiona cualquiera de los botones = Infracción
    - El sistema activa inmediatamente el castigo de electrochoque
    - El electrochoque solo se detiene cuando AMBOS botones NO están presionados
3. Sistema de castigo
    - 🔴 Castigo instantáneo**: El electrochoque comienza en el momento en que el talón presiona el botón
    - 📈 Intensidad progresiva**: Aumenta automáticamente con cada infracción
    - ⏱️ Castigo continuo**: Mientras un botón esté presionado, el electrochoque continúa hasta que se libere completamente
    - 🎮 Castigo dilema: Después de estar de puntillas un tiempo, se activa el dispositivo TD01 para crear interferencia
4. Fin del juego
    - Se desbloquea automáticamente al alcanzar el tiempo establecido
    - El sistema muestra un informe de desempeño completo
    - Estadísticas del número de infracciones, electrochoques aplicados, etc. 💪 Puntos Clave del Desafío
+ Desafío físico**: Mantenerse de puntillas durante mucho tiempo pone a prueba la resistencia muscular de las piernas
+ Prueba de voluntad**: Mantener la postura estándar frente al cansancio
+ Control preciso**: Cualquier mínimo descenso del talón será detectado
+ Juego psicológico**: Mantener la calma y la concentración bajo la amenaza del electrochoque 🏆 Condiciones de Victoria
¡Sobrevive hasta el final del tiempo establecido, menos infracciones es mejor desempeño!
+ 🌟 Desempeño perfecto**: 0 infracciones
+ ⭐ Desempeño excelente**: 1-2 infracciones
+ 👍 Buen desempeño**: 3-5 infracciones
+ 😅 Necesita mejorar**: Más de 5 infracciones

+ Se recomienda comenzar con intensidad baja y tiempo corto
