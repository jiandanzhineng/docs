# Juego de Castigo en Puntas de Pie v2

Juego de entrenamiento de autocontrol corporal de alta tecnología - Desafío de permanecer en puntas de pie

### Contenido actualizado en V1.4.0:
1. **Agregado dispositivo TD01**: Evita largos períodos de aburrimiento, se activa después de un tiempo sin recibir descargas eléctricas, con intensidad gradualmente creciente
2. **Agregada operación de inicio del juego mediante botón en zidongsuo**: Facilita la operación individual; una vez activado el modo de inicio manual, el juego puede comenzar mediante un botón
3. **zidongsuo y TD01 son dispositivos opcionales**: El juego puede jugarse normalmente sin estos dispositivos, solo se necesita el dispositivo de descargas eléctricas y el sensor QTZ

## 🎮 Explicación detallada del juego
### 📋 Preparación del juego
#### 1. Configuración del equipo
El sistema conecta automáticamente cuatro dispositivos principales:

+ 🔒 **Cerradura automática**: Se bloquea al inicio del juego, evitando abandonos durante la partida (dispositivo opcional)
+ ⚡ **Dispositivo de descargas eléctricas**: Proporciona castigo de retroalimentación inmediata
+ 📡 **Sensor láser QTZ**: Monitorea si el talón está presionando el suelo a través de dos botones (colocado debajo del talón)
+ 🎮 **Dispositivo TD01**: Dispositivo de refuerzo opcional, se activa después de un tiempo sin recibir descargas eléctricas, aumentando gradualmente la intensidad para perturbar mantenerse en puntas de pie.

#### 2. Configuración de parámetros
La dificultad del juego es personalizable:

+ **Duración del juego**: Ajustable de 1 a 360 minutos
+ **Intensidad de descarga**: Niveles de 10 a 100
+ **Modo progresivo**: Cuantas más infracciones, más severo el castigo
+ **Límite de seguridad**: Opción para permitir o no superar el límite seguro de 30V
+ **Inicio manual**: Opción para requerir un clic en el botón del dispositivo auto_lock para comenzar el juego
+ **Retardo del TD01**: Ajustable de 1 a 60 segundos, tiempo sin descargas antes de activar el TD01
+ **Incremento de intensidad del TD01**: Valor de intensidad agregado cada 5 segundos (1-100)

## 🎯 Cómo jugar específicamente
### Reglas principales
Mantén la postura de estar en puntas de pie. ¡El talón nunca debe tocar el suelo!

#### 1. Inicio del juego
+ El sistema se bloquea automáticamente (si hay cerradura automática), no se puede abandonar durante la partida
+ Si está activado el modo de inicio manual, debes hacer clic en el botón del dispositivo auto_lock para comenzar el temporizador
+ Comienza el conteo de tiempo, debes mantener la postura en puntas de pie
+ El sensor QTZ comienza a monitorear en tiempo real la posición del talón

#### 2. Mecanismo de monitoreo
+ El sensor tiene dos puntos de detección de botones (button0 y button1)
+ Presionar cualquiera de los botones con el talón = infracción
+ El sistema activa inmediatamente el castigo de descarga eléctrica
+ La descarga solo se detiene cuando ambos botones no están presionados

#### 3. Sistema de castigo
+ 🔴 **Castigo inmediato**: La descarga eléctrica comienza en el instante en que el talón presiona el botón
+ 📈 **Intensidad progresiva**: Cuantas más infracciones, mayor la intensidad de la descarga automáticamente
+ ⏱️ **Castigo continuo**: La descarga continúa mientras el botón esté presionado, hasta que se levante completamente
+ 🎮 **Castigo reforzado con TD01**: El dispositivo TD01 se activa después de un tiempo especificado sin recibir descargas eléctricas, con intensidad incrementada cada 5 segundos

#### 4. Mecanismo del dispositivo TD01
+ **Activación retardada**: Se activa automáticamente después de una cantidad especificada de segundos sin recibir descargas eléctricas
+ **Incremento de intensidad**: La intensidad aumenta automáticamente cada 5 segundos según el valor configurado
+ **Intensidad máxima**: Soporta un rango de 0-255, no afectado por límites de seguridad
+ **Parada automática**: Se detiene y reinicia automáticamente cuando se activa una descarga eléctrica

#### 5. Fin del juego
+ Se desbloquea automáticamente después de llegar al tiempo establecido
+ El sistema muestra un informe completo de rendimiento
+ Estadísticas del número de infracciones, descargas eléctricas, etc.

## 💪 Puntos clave del desafío
+ **Desafío físico**: Mantenerse en puntas de pie por mucho tiempo prueba la resistencia muscular de las piernas
+ **Prueba de fuerza de voluntad**: Mantener la postura estándar frente al cansancio
+ **Control preciso**: Cualquier hundimiento mínimo del talón será detectado
+ **Juego psicológico**: Mantener la calma y concentración bajo la disuasión de las descargas eléctricas
+ **Doble presión**: Evitar descargas eléctricas y lidiar con la interferencia del dispositivo TD01

## 🏆 Condiciones de victoria
Lograr mantenerse hasta el final del tiempo establecido. Cuantas menos infracciones, mejor el rendimiento:

+ 🌟 **Rendimiento perfecto**: 0 infracciones
+ ⭐ **Rendimiento excelente**: 1-2 infracciones
+ 👍 **Rendimiento bueno**: 3-5 infracciones
+ 😅 **Necesita mejorar**: Más de 5 infracciones

## ⚠️ Recordatorio de seguridad
+ Asegúrate de estar sano, sin enfermedades cardíacas, epilepsia, etc.
+ Se recomienda comenzar con menor intensidad y tiempos más cortos
+ Deja el juego inmediatamente si sientes malestar durante la partida
+ Solo para adultos mayores de 18 años
+ Configura la intensidad de descarga con precaución. Un valor demasiado alto puede causar lesiones por electrocución, el riesgo es responsabilidad propia.
+ Si activas la función de exceder el límite de seguridad, asumes el riesgo
+ Asegura un entorno de juego seguro, se recomienda supervisión

## 🔧 Funciones especiales
### Modo de inicio manual
+ Muestra una pantalla de espera después de cargar el juego
+ La cerradura automática se bloquea primero
+ Requiere un clic en el botón del dispositivo auto_lock para comenzar el temporizador del juego
+ Ideal para situaciones que requieren tiempo de preparación

### Mecanismo de descarga en tiempo real
+ Comienza la descarga eléctrica continua inmediatamente al presionar cualquier botón
+ Detiene inmediatamente la descarga eléctrica cuando todos los botones se liberan
+ Proporciona el mecanismo de retroalimentación más directo

### Sistema de refuerzo TD01
+ Dispositivo de desafío adicional opcional
+ Aumenta la dificultad cuando el rendimiento es bueno
+ La intensidad aumenta gradualmente, incrementando la tensión del juego
+ Soporta control de intensidad independiente
