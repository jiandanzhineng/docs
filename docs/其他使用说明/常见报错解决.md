# 常见报错解决

1. 设备配网后，客户端启动后无法自动上线
    1. 请确认您的路由器是否支持 mdns，参考[检测路由器是否支持 mdns（大部分路由器都支持）](./检测路由器是否支持mdns（大部分路由器都支持）.md) 如果提示不支持可以
        1. 通过手机热点连接电脑和设备进行游玩
        2. 检查您的路由器和光猫连接关系，如果路由器模式为AP，可以修改为DHCP模式
2. vcruntime140.dll相关报错

[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)

下载安装<font>Visual C++ Redistributable for Visual Studio 2015</font>
