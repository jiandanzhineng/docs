# 使用流程
1. 下载termux 下载地址：[https://update.ezsapi.top/termux.apk](https://update.ezsapi.top/termux.apk)
2. 安装termux 并将termux的电源管理设置为**<font>不限制 后台锁定</font>**防止被杀后台
3. 打开termux

**<font>初次运行，请运行下面的语句，复制贴进termux 按回车</font>**

```bash
curl -fsSL https://update.ezsapi.top/run_control_panel.sh | sh
```

如**<font>非初次运行</font>** 可运行下面的语句 以提升加载速度（如果想更新的话还是用上面的）

```bash
cd "${HOME}/control-panel-stable" && npm run dev:all
```

等待运行完成 类似下图（如20分钟后还没有类似图像 请参考常见问题1）

![1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663.jpeg](./img/aeI6B3lspG60kJMv/1761797537885-6667e5d2-a8ea-4d3b-956d-32f8ebe1d663-159999.jpeg)

4. 打开手机热点 建议将热点名设置为easysmart 密码设置为8个1 设备会自动连接 如果不是这个请参考其他使用说明中的设备配网方式进行配网，建议用默认选项
5. 在手机浏览器打开访问 127.0.0.1:5173
6. 此时 在设备管理应该可以看到您的设备已经连接 在游戏列表中可以启动游戏（如为空可以刷新一下）

# 常见问题
## 没有正常启动
由于网络问题，存在一定概率启动失败此时有两个解决方案

1. 重新安装termux（或者清除termux全部数据），之后重新运行
2. 重启手机，重新安装termux（或者清除termux全部数据）。之后再次运行
3. 如果是下载网速过慢 可以避开晚上7点到12点时间段进行下载，这段时间网络通常比较拥堵。上午一般比较通畅。

另外还要检查是否设置了termux<font>后台无限制</font>

如果尝试以上两种方式还是无法解决，请加QQ群 970326066 联系群主处理~

## 设备无法连接到手机热点
请将手机热点切换为2.4G 如果还不行请用其他设备连接这个手机的热点试试
可能原因1：出现过其他手机也连不上手机热点的情况，说明手机热点有问题。
可能原因2：还可能是手机热点问题 原因似乎是因为热点始终无法切换到2.4G，一位用户通过将路由器上的聚合频段分成2.4和5两个解决了，解决原因未知。

可能原因3：用户进行了配网操作，设备不再连接默认wifi，此时请用配网APP将设备重新配网到默认wifi和密码，APP的配网wifi名是可以修改的
