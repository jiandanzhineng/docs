# 锥刺股背单词玩法说明

# 玩法说明：
背单词的时候总是很困

这时候需要亿点点刺激来消除困意

只需将该设备放到身上，就可以在你背错的时候给你提提精神~

这样效率就会直线上升啦

# 使用前提
1. 家里有2.4Gwifi 
2. wifi和电脑是一个局域网（是同一个路由器）
3. 路由器支持 mdns（如不支持需要用手机热点）[检测路由器是否支持 mdns（大部分路由器都支持）](../其他使用说明/检测路由器是否支持mdns（大部分路由器都支持）.md)
4. 已有简单智能脉冲终端：[简单智能脉冲（电击）终端](../设备介绍&购买/简单智能脉冲（电击）终端.md) 或 [淘宝链接](https://item.taobao.com/item.htm?id=892309919507)

## 设备配网
（首次使用前建议先给设备充电）

1. 拨动开关开启设备 开启后设备亮灯
2. 启动小程序 给设备配网

此步骤请看[通过 APP 将设备连接到 WiFi](../其他使用说明/设备连接wifi（配网）/通过APP将设备连接到wifi.md) 或 [通过小程序将设备连接到 WiFi](../其他使用说明/设备连接wifi（配网）/通过小程序将设备连接到wifi.md)

## 在电脑上运行服务端
1. 电脑上运行 步骤2启动.bat 运行后2分钟左右启动完成（首次运行时间比较长） 出现红框内容说明启动成功（无该文件请在文档头部下载）
