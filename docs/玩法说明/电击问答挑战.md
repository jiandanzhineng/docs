# 电击问答挑战

## 游戏简介
问答游戏是一个支持外部题目导入的答题玩法，玩家需要在规定时间内回答问题。答对进入下一题，答错会受到电击惩罚并将题目放到后面重新回答。

## 游戏特色
+ 📚 **自定义题库**：支持导入外部JSON格式的题目文件
+ ⚡ **电击惩罚**：答错题目会触发电击设备进行惩罚
+ ⏰ **时间限制**：每题都有答题时间限制，超时视为答错
+ 🎮 **多种操作方式**：支持鼠标点击和QTZ按键设备操作
+ 🔒 **手动开启**：可选择手动开启模式，通过自动锁设备按键启动游戏

## 设备要求
### 必需设备
+ **电击设备 (DIANJI)**：用于答错时执行电击惩罚

### 可选设备
+ **自动锁设备 (ZIDONGSUO)**：用于手动开启游戏功能
+ **QTZ按键设备 (QTZ)**：可代替鼠标进行答案选择

## 游戏参数配置
### 题目设置
+ **题目文件路径**：指定包含题目的JSON文件路径
    - 默认：`<OUTTER_GAME>/QA-game/女仆行为规范考核题库.json`
    - 支持`<OUTTER_GAME>`标记指向打包后的outter-game文件夹

### 惩罚设置
+ **电击强度**：答错时的电击强度 (10-100)
+ **电击持续时间**：电击持续时间 (1-10秒)

### 时间设置
+ **答题时间限制**：每题的答题时间限制 (10-300秒，0为无限制)

### 启动设置
+ **手动开启**：启用后需要通过自动锁设备按键才能开始游戏

## 题目文件格式
题目文件应为JSON格式，包含以下结构：

```json
{
  "questions": [
    {
      "question": "题目内容",
      "options": ["选项A", "选项B", "选项C", "选项D"],
      "correct": 0,
      "explanation": "答案解释（可选）"
    }
  ]
}
```

### 字段说明
+ `question`：题目文本
+ `options`：选项数组
+ `correct`：正确答案的索引（从0开始）
+ `explanation`：答案解释（可选）

## 游戏流程
1. **游戏准备**
    - 加载题目文件
    - 检查设备连接状态
    - 初始化游戏参数
2. **游戏开始**
    - 手动开启模式：等待自动锁设备按键信号
    - 自动开始模式：立即开始第一题
3. **答题过程**
    - 显示题目和选项
    - 自动锁设备锁定（防止中途退出）
    - 开始倒计时
    - 等待玩家选择答案
4. **答案判定**
    - **答对**：显示正确提示，进入下一题
    - **答错或超时**：触发电击惩罚，题目放到队列末尾重新回答
5. **游戏结束**
    - 所有题目答对后游戏结束
    - 自动锁设备解锁
    - 显示游戏统计信息

## 操作方式
### 鼠标操作
+ 点击选项按钮选择答案

### QTZ按键操作
+ **按键0**：向上切换选项
+ **按键1**：向下切换选项 / 确认选择
+ 选中的选项会高亮显示

## 注意事项
1. 确保电击设备已正确连接并配置，电压强度请自行调节，因强度过高导致的电击伤害需自行承担
2. 自动锁由于网络条件等因素，可能存在不稳定情况，务必准备脱困道具。
