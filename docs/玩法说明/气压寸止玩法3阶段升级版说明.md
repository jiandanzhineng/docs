## 简介
这是一个基于气压传感器的寸止训练玩法。系统根据气压值智能调节震动强度，引导用户保持在兴奋边缘。当气压超过临界值时，会触发电击惩罚。

## 游戏流程
游戏包含以下几个状态循环：

![流程示意图](./img/cunzhi3phase.jpg)

1. **平静期 (Initial/Sub Calm)**
   - 震动强度缓慢上升，伴随随机扰动。
   - **目标**：提升兴奋度，增加气压。
   - **跳转**：气压超过「中间压力」进入中期。

2. **中期 (Middle)**
   - 智能控制：气压越高，震动强度越低（防止过快爆发）；气压越低，震动强度越高（维持兴奋）。
   - **跳转**：
     - 气压 >= 「临界气压」：触发**电击**，进入边缘期。
     - 气压 < 「中间压力」：回落至平静期。

3. **边缘期 (Edging)**
   - **触发电击**！震动停止。
   - **跳转**：气压降回临界值以下后，进入冷却延迟。

4. **延迟期 (Delay)**
   - 等待一段时间（冷静延迟）。
   - **跳转**：延迟结束后，根据当前气压决定是回到中期（大于中间压力）还是平静期（小于中间压力）。



## 所需设备
1. **气压传感器 (QIYA)**：必须。用于监测兴奋程度。
2. **偏轴电机控制器 (TD01)**：必须。用于提供震动刺激。
3. **电击设备 (DIANJI)**：可选。用于边缘惩罚。
4. **自动锁 (ZIDONGSUO)**：可选。游戏开始自动上锁，结束解锁。


## 使用说明

1. 完成气压设备组装，将所有设备连接到网络。
设备组装视频教程：https://youtu.be/QQaT823gKHc
2. 将气压设备放入身体，一般充气至20-25kpa，确保轻压时压力有明显变化的情况下压力适当低一些，这样可以减缓漏气。
3. 设置中间气压和临界气压，初次玩的时候可以边玩边调整，这两个气压忍不住了就降低一些，没感觉提升一些。

## 配置说明

| 参数名称 | 说明 | 推荐值 |
| :--- | :--- | :--- |
| **游戏时长** | 游戏运行的总时间（分钟）。 | 20 |
| **临界气压** | 触发电击和寸止的最高气压阈值 (kPa)。 | 23 |
| **TD01最大强度** | 震动的最大力度 (1-255)。 | 255 |
| **中间(兴奋)压力** | 进入智能控制阶段的压力门槛 (kPa)。 | 22 |
| **电击强度** | 惩罚时的电压强度 (V)。 | 20 |
| **电击持续** | 惩罚持续时间 (秒)。 | 3 |
| **低压延迟** | 触发寸止后的冷却休息时间 (秒)。 | 5 |
| **强度递增速率上限** | 震动增强的最快速度，避免突变。 | 10 |
| **强度随机扰动** | 震动强度的随机波动幅度 (%)。 | 0 |
| **延迟后逐步提升** | 平静期震动每秒自动增加的数值。 | 10 |
