# 提肛训练玩法

# 玩法简介
+ 提肛训练通过“放松阶段”和“提肛阶段”交替进行
+ 目标是在规定时间内完成尽可能多的“提肛成功”
+ 成功或失败会在界面直观提示，失败会触发电击（如已连接并启用）

# 设备与准备
+ 必需设备：`气压传感器（QIYA）`
+ 可选设备：`电击设备（DIANJI）`、`自动锁（ZIDONGSUO）`
+ 开始自动上锁，结束自动解锁（若连接了自动锁）

# 参数说明
+ `持续时间(分钟)`：整局游戏的时长
+ `提肛次数目标`：期望完成的成功次数，用于显示总进度
+ `气压变化度(kPa)`：提肛阶段需要达到的气压增量阈值（相对于放松阶段的最低气压）
+ `电击强度(V)`：失败时的电击强度
+ `电击持续时间(秒)`：失败时电击持续的时间
+ `单个循环时间(秒)`：每个阶段的时长，默认 10 秒（放松 10 秒 → 提肛 10 秒 → 循环）

# 玩法流程
+ 放松阶段
    - 放松并自然呼吸，系统会记录本阶段的“最低气压”作为参考
    - 倒计时结束后进入提肛阶段
+ 提肛阶段
    - 需要在本阶段时长内，将气压提升到“最低气压 + 气压变化度”
    - 达到目标即判定为“已达成”，但仍需等本阶段倒计时结束才进入下一轮
    - 若阶段结束前未达标，判定“挑战失败 · 开始电击”（如电击设备可用）
+ 阶段交替
    - 每轮按“放松 → 提肛 → 放松 → …”循环，直至时间结束或达到目标次数

# 界面提示
+ 顶部大字显示当前阶段（放松阶段/提肛阶段）和本阶段剩余时间
+ 总体进度
    - 按次数的进度条：已完成次数 / 目标次数
    - 按时间的进度条：已用时间 / 总时长
+ 压力与目标
    - 当前压力（kPa）
    - 放松期最低压力（参考值）
    - 提肛目标压力（最低压力 + 气压变化度）
+ 成功/失败提示
    - 成功：数字后显示绿色胶囊“已达成”
    - 失败：数字后显示红色胶囊“挑战失败 · 开始电击”
+ 操作与日志
    - 按钮：暂停、手动电击
    - 日志：显示最近信息与系统提示

**结束与统计**

+ 到达设定的持续时间或成功次数达到目标时结束
+ 结束后自动解锁（若已连接自动锁）
+ 界面显示累计成功次数与电击次数等

**使用建议**

+ 初次使用建议将“气压变化度”设置较低，以熟悉节奏
+ 若连接电击设备，请从低强度开始并逐步调节
+ 保持呼吸平稳，提肛阶段专注提升压力到达目标

**快速上手**

+ 连接气压传感器（可选连接电击、自动锁）
+ 设置参数并开始
+ 放松阶段无需用力，等待倒计时；提肛阶段用力提升到目标
+ 看到“已达成”即可保持直到本阶段结束；失败则会提示并电击
+ 重复循环直至结束，查看统计与进度条了解完成情况
