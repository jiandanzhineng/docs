+ 参数设置 ：可自定义游戏难度
    - 游戏时长：1-180分钟可调
    - 电击强度：10-100级别
    - 渐进模式：违规次数越多，惩罚越重 🎯 具体玩法

核心规则 ：保持踮脚站立姿态，脚后跟绝对不能接触地面！

1. 游戏开始
    - 系统自动锁定，无法中途退出
    - 开始计时，需要保持踮脚姿态
    - QTZ传感器开始实时监控脚后跟位置
2. 监控机制
    - 传感器有两个按钮检测点
    - 脚后跟一旦压下任意一个按钮 = 违规
    - 系统立即触发电击惩罚
    - 只有当两个按钮都未被按下时，电击才会停止
3. 惩罚系统
    - 🔴 即时惩罚 ：脚后跟压下按钮瞬间开始电击
    - 📈 渐进强度 ：违规次数越多，电击强度自动递增
    - ⏱️ 持续惩罚 ：只要按钮被压下就持续电击，直到完全抬起
    - 🎮 两难惩罚：在踮脚一段时间后启动TD01设备，进行干扰
4. 游戏结束
    - 坚持到设定时间后自动解锁
    - 系统显示完整的表现报告
    - 统计违规次数、电击次数等数据 💪 挑战要点
+ 体力挑战 ：长时间踮脚站立考验腿部肌肉耐力
+ 意志力考验 ：面对疲劳时仍需保持标准姿态
+ 精准控制 ：任何细微的脚后跟下沉都会被检测到
+ 心理博弈 ：在电击威慑下保持冷静和专注 🏆 胜利条件  
成功坚持到设定时间结束，违规次数越少表现越优秀：
+ 🌟 完美表现 ：0次违规
+ ⭐ 优秀表现 ：1-2次违规
+ 👍 良好表现 ：3-5次违规
+ 😅 需要改进 ：5次以上违规

+ 游戏过程中如有不适请立即停止
