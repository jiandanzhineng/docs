# 踮脚惩罚玩法 v2

高科技体感自律训练游戏 - 踮脚站立挑战

1. **增加TD01设备**：防止长时间无趣，在未触发电击一定时间后启动，强度逐渐递增
2. **增加zidongsuo按键开始游戏操作**：方便单人操作，启用手动开启模式后可通过按键开始游戏

## 🎮 游戏玩法详解
### 📋 游戏准备
#### 1. 设备配置
系统会自动连接四个核心设备：

+ 🔒 **自动锁**：游戏开始时锁定，防止中途退出（可选设备）
+ ⚡ **电击设备**：提供即时反馈惩罚
+ 📡 **QTZ激光传感器**：通过双按钮监测脚后跟是否踩下（放置在脚后跟下方）
+ 🎮 **TD01设备**：可选增强设备，在未触发电击一定时间后启动，逐渐增加强度，干扰踮脚。

#### 2. 参数设置
可自定义游戏难度：

+ **游戏时长**：1-360分钟可调
+ **电击强度**：10-100级别
+ **渐进模式**：违规次数越多，惩罚越重
+ **安全限度**：可选择是否允许超过30V安全限度
+ **手动开启**：可选择等待auto_lock设备按键点击才开始游戏
+ **TD01延时**：1-60秒可调，未触发电击多久后启动TD01
+ **TD01强度增幅**：每5秒增加的强度值（1-100）

## 🎯 具体玩法
### 核心规则
保持踮脚站立姿态，脚后跟绝对不能接触地面！

#### 1. 游戏开始
+ 系统自动锁定（如有自动锁设备），无法中途退出
+ 如启用手动开启模式，需点击auto_lock设备按键开始计时
+ 开始计时，需要保持踮脚姿态
+ QTZ传感器开始实时监控脚后跟位置

#### 2. 监控机制
+ 传感器有两个按钮检测点（button0和button1）
+ 脚后跟一旦压下任意一个按钮 = 违规
+ 系统立即触发电击惩罚
+ 只有当两个按钮都未被按下时，电击才会停止

#### 3. 惩罚系统
+ 🔴 **即时惩罚**：脚后跟压下按钮瞬间开始电击
+ 📈 **渐进强度**：违规次数越多，电击强度自动递增
+ ⏱️ **持续惩罚**：只要按钮被压下就持续电击，直到完全抬起
+ 🎮 **TD01增强惩罚**：在未触发电击指定时间后启动TD01设备，强度每5秒递增

#### 4. TD01设备机制
+ **延时启动**：在未触发电击指定秒数后自动启动
+ **强度递增**：每5秒自动增加设定的强度值
+ **最大强度**：支持0-255范围，不受安全限度影响
+ **自动停止**：当触发电击时会自动停止并重置

#### 5. 游戏结束
+ 坚持到设定时间后自动解锁
+ 系统显示完整的表现报告
+ 统计违规次数、电击次数等数据

## 💪 挑战要点
+ **体力挑战**：长时间踮脚站立考验腿部肌肉耐力
+ **意志力考验**：面对疲劳时仍需保持标准姿态
+ **精准控制**：任何细微的脚后跟下沉都会被检测到
+ **心理博弈**：在电击威慑下保持冷静和专注
+ **双重压力**：既要避免电击，又要应对TD01设备的干扰

## 🏆 胜利条件
成功坚持到设定时间结束，违规次数越少表现越优秀：

+ 🌟 **完美表现**：0次违规
+ ⭐ **优秀表现**：1-2次违规
+ 👍 **良好表现**：3-5次违规
+ 😅 **需要改进**：5次以上违规

## ⚠️ 安全提醒
+ 请确保身体健康，无心脏病、癫痫等疾病
+ 建议从较低强度和较短时间开始

+ 仅限18岁以上成年人使用
+ 电击强度请谨慎设置，过高可能导致电击伤害，风险需自行承担
+ 如启用超过安全限度功能，风险自负
+ 确保游戏环境安全，建议有人监督
